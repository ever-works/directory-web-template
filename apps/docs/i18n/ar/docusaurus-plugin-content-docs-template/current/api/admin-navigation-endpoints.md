# التنقل الإداري ونقاط نهاية فهرس الموقع

تقوم نقاط النهاية الإدارية هذه بإدارة روابط التنقل المخصصة للموقع وفهرس الموقع الجغرافي. تسمح نقاط نهاية التنقل بتكوين روابط الرأس والتذييل المخصصة المخزنة في `config.yml`. تقوم نقاط نهاية فهرس الموقع بإدارة الفهرس المكاني المستخدم للتحليلات الجغرافية وميزات الخريطة.

## نظرة عامة

|نقطة النهاية|الطريقة|مصادقة|الوصف|
|---|---|---|---|
|`/api/admin/navigation`|احصل على|المشرف|احصل على تكوين التنقل المخصص|
|`/api/admin/navigation`|التصحيح|المشرف|تحديث عناصر التنقل المخصصة|
|`/api/admin/location-index`|احصل على|المشرف|الحصول على إحصائيات فهرس الموقع|
|`/api/admin/location-index`|بريد|المشرف|إعادة بناء أو مسح فهرس الموقع|

## نقاط نهاية الملاحة

### الحصول على تكوين الملاحة

```
GET /api/admin/navigation
```

يسترد عناصر التنقل `custom_header` و`custom_footer` من ملف `config.yml` الخاص بالموقع. إرجاع صفائف فارغة إذا لم يتم تكوين التنقل المخصص.

**المصادقة:** مطلوب مشرف (عبر `getCachedApiSession`)

**رد النجاح (200):**

```json
{
  "custom_header": [
    {
      "label": "About",
      "path": "/about"
    },
    {
      "label": "Documentation",
      "path": "/pages/docs"
    }
  ],
  "custom_footer": [
    {
      "label": "GitHub",
      "path": "https://github.com/example"
    },
    {
      "label": "footer.PRIVACY_POLICY",
      "path": "/pages/privacy-policy"
    }
  ]
}
```

يحتوي كل عنصر تنقل على حقلين:

|الميدان|اكتب|الوصف|
|---|---|---|
|`label`|سلسلة|عرض النص (نص عادي أو مفتاح ترجمة i18n مثل `"footer.PRIVACY_POLICY"`)|
|`path`|سلسلة|مسار URL (مسار داخلي يبدأ بـ `/` أو عنوان URL خارجي بـ `http://`/`https://`)|

|الحالة|الحالة|
|---|---|
| 401 |لم يتم المصادقة عليه كمسؤول|
| 500 |فشل في قراءة التكوين|

**المصدر:** `template/app/api/admin/navigation/route.ts`

### تحديث تكوين التنقل

```
PATCH /api/admin/navigation
```

يقوم بتحديث عناصر التنقل المخصصة للرأس أو التذييل في `config.yml`. يتحقق من صحة تنسيق مسار كل عنصر لمنع هجمات XSS عبر مخططات عناوين URL الخطيرة.

**المصادقة:** مطلوب المشرف

**نص الطلب:**

```json
{
  "type": "header",
  "items": [
    {
      "label": "About",
      "path": "/about"
    },
    {
      "label": "Blog",
      "path": "https://blog.example.com"
    }
  ]
}
```

|الميدان|اكتب|مطلوب|الوصف|
|---|---|---|---|
|`type`|سلسلة|نعم|`"header"` أو `"footer"`|
|`items`|صفيف|نعم|مجموعة من عناصر التنقل|
|`items[].label`|سلسلة|نعم|تسمية العرض غير فارغة|
|`items[].path`|سلسلة|نعم|مسار URL صالح|

**التحقق من صحة المسار:**

تفرض الدالة `isValidNavigationPath()` قواعد صارمة لتنسيق المسار:

|تنسيق المسار|مسموح|مثال|
|---|---|---|
|الطرق الداخلية|نعم|`/about`، `/pages/docs`|
|عناوين URL لـ HTTPS|نعم|`https://example.com`|
|عناوين URL HTTP|نعم|`http://example.com`|
|عناوين URL المتعلقة بالبروتوكول|لا|`//evil.com`|
|عناوين URL لجافا سكريبت|لا|`javascript:alert(1)`|
|عناوين URL للبيانات|لا|`data:text/html,...`|
|مخططات أخرى|لا|`vbscript:`، `file:`|

**رد النجاح (200):**

```json
{
  "success": true,
  "type": "header",
  "items": [
    {
      "label": "About",
      "path": "/about"
    }
  ]
}
```

** ردود الأخطاء: **

|الحالة|الحالة|
|---|---|
| 400 |`type` ليس `"header"` أو `"footer"`|
| 400 |`items` ليس مصفوفة|
| 400 |العنصر مفقود `label` أو `path` حقول السلسلة|
| 400 |تنسيق مسار غير صالح (منع XSS)|
| 401 |لم يتم المصادقة عليه كمسؤول|
| 500 |فشل في كتابة التكوين|

قم بتمرير مصفوفة `items` فارغة لمسح كل التنقل المخصص للنوع المحدد.

**المصدر:** `template/app/api/admin/navigation/route.ts`

## نقاط نهاية فهرس الموقع

### احصل على إحصائيات فهرس الموقع

```
GET /api/admin/location-index
```

إرجاع إحصائيات حول فهرس الموقع الجغرافي بما في ذلك إجمالي العناصر المفهرسة وعدد المدن والدول وإعادة بناء البيانات التعريفية. يستخدم خدمة فهرس الموقع لاسترجاع البيانات.

**المصادقة:** مطلوب مشرف (عبر `checkAdminAuth()`)

**التخزين المؤقت:** معطل - يستخدم `force-dynamic` و`revalidate: 0` و`force-no-store`.

**رد النجاح (200):**

```json
{
  "success": true,
  "data": {
    "totalIndexed": 450,
    "citiesCount": 85,
    "countriesCount": 25,
    "remoteCount": 30,
    "lastIndexedAt": "2024-01-20T10:30:00.000Z",
    "lastRebuildAt": "2024-01-15T08:00:00.000Z"
  }
}
```

|الحالة|الحالة|
|---|---|
| 401 |لم يتم المصادقة عليه كمسؤول|
| 500 |خطأ داخلي في الخادم|

**المصدر:** `template/app/api/admin/location-index/route.ts`

### إدارة فهرس الموقع

```
POST /api/admin/location-index
```

ينفذ إجراءات الإدارة على فهرس الموقع. يدعم إعادة بناء الفهرس من البداية أو مسح كافة الإدخالات.

**المصادقة:** مطلوب المشرف

**نص الطلب:**

```json
{
  "action": "rebuild"
}
```

|الميدان|اكتب|مطلوب|الوصف|
|---|---|---|---|
|`action`|سلسلة|نعم|`"rebuild"` أو `"clear"`|

**الإجراءات:**

|العمل|الوصف|
|---|---|
|`rebuild`|جلب جميع العناصر من المستودع وإعادة فهرسة بيانات موقعها. إرجاع إعادة بناء الإحصائيات.|
|`clear`|إزالة كافة الإدخالات من فهرس الموقع. إرجاع عدد الإدخالات التي تم مسحها.|

**استجابة النجاح (200)--إعادة البناء:**

```json
{
  "success": true,
  "data": {
    "indexed": 420,
    "skipped": 80,
    "errors": 0
  }
}
```

**استجابة النجاح (200)--مسح:**

```json
{
  "success": true,
  "data": {
    "cleared": 450
  }
}
```

** ردود الأخطاء: **

|الحالة|الحالة|
|---|---|
| 400 |إجراء غير صالح (ليس `"rebuild"` أو `"clear"`)|
| 401 |لم يتم المصادقة عليه كمسؤول|
| 500 |خطأ داخلي في الخادم|

**المصدر:** `template/app/api/admin/location-index/route.ts`

## تفاصيل التنفيذ الرئيسية

- **منع XSS:** يرفض التحقق من صحة مسار التنقل جميع أنظمة URL باستثناء `/`، و`http://`، و`https://`. يؤدي هذا إلى حظر `javascript:` و`data:` و`vbscript:` وعناوين URL المتعلقة بالبروتوكول (`//evil.com`) التي يمكن استخدامها للبرمجة النصية عبر المواقع.
- ** تخزين التكوين: ** يتم تخزين عناصر التنقل في `config.yml` ضمن `custom_header` و`custom_footer`، ويتم الاحتفاظ بها عبر `configManager.updateNestedKey()`.
- **ملصقات i18n:** يمكن أن تكون تسميات التنقل نصًا عاديًا أو مفاتيح ترجمة (على سبيل المثال، `"footer.PRIVACY_POLICY"`). الواجهة الأمامية مسؤولة عن حل مفاتيح الترجمة.
- **إعادة بناء فهرس الموقع:** تقوم عملية إعادة البناء بتحميل كافة العناصر من `ItemRepository` وتمريرها إلى خدمة فهرس الموقع. يمكن أن تكون هذه عملية كثيفة الاستخدام للموارد لمجموعات البيانات الكبيرة.
- **انتهاك ذاكرة التخزين المؤقت:** تقوم نقاط نهاية فهرس الموقع بتعطيل جميع عمليات التخزين المؤقت بشكل صريح لضمان أن لوحة تحكم المشرف تعرض البيانات الحالية دائمًا.
