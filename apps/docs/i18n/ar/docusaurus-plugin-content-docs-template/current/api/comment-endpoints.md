# نقاط نهاية التعليق

يوفر نظام التعليقات نقاط نهاية لإنشاء التعليقات على العناصر وقراءتها وتحديثها وحذفها. تتضمن التعليقات تصنيفًا من 1 إلى 5 نجوم وتدعم الوصول العام (القراءة) والعمليات الموثقة (الإنشاء/التحرير/الحذف). توفر نقاط النهاية الإدارية إمكانات الإشراف.

## نظرة عامة

### نقاط النهاية العامة

|نقطة النهاية|الطريقة|مصادقة|الوصف|
|---|---|---|---|
|`/api/items/[slug]/comments`|احصل على|عام|قائمة التعليقات لأحد العناصر|
|`/api/items/[slug]/comments/rating`|احصل على|عام|الحصول على إحصائيات التصنيف الإجمالية|
|`/api/items/[slug]/comments/rating/[commentId]`|احصل على|عام|الحصول على تصنيف تعليق واحد|

### نقاط النهاية المصادق عليها

|نقطة النهاية|الطريقة|مصادقة|الوصف|
|---|---|---|---|
|`/api/items/[slug]/comments`|بريد|المستخدم|إنشاء تعليق جديد|
|`/api/items/[slug]/comments/[commentId]`|وضع|مالك|تحديث التعليق الخاص|
|`/api/items/[slug]/comments/[commentId]`|حذف|مالك|حذف التعليق الخاص|
|`/api/items/[slug]/comments/rating/[commentId]`|التصحيح|المستخدم|تحديث تقييم التعليق|

### نقاط نهاية المشرف

|نقطة النهاية|الطريقة|مصادقة|الوصف|
|---|---|---|---|
|`/api/admin/comments`|احصل على|المشرف|قائمة كافة التعليقات مع ترقيم الصفحات|
|`/api/admin/comments/[id]`|احصل على|المشرف|الحصول على تعليق بواسطة معرف|
|`/api/admin/comments/[id]`|وضع|المشرف|تحديث محتوى التعليق|
|`/api/admin/comments/[id]`|حذف|المشرف|حذف التعليق بشكل مبسط|

## نقاط النهاية العامة

### تعليقات عناصر القائمة

```
GET /api/items/[slug]/comments
```

إرجاع كافة التعليقات لعنصر معين بما في ذلك معلومات ملف تعريف المستخدم. لا يلزم المصادقة.

** معلمات المسار: **

|المعلمة|اكتب|الوصف|
|---|---|---|
|`slug`|سلسلة|سبيكة البند|

**رد النجاح (200):**

```json
{
  "success": true,
  "comments": [
    {
      "id": "comment_123abc",
      "content": "This is an amazing tool!",
      "rating": 5,
      "userId": "client_456def",
      "itemId": "item_123abc",
      "createdAt": "2024-01-20T10:30:00.000Z",
      "updatedAt": "2024-01-20T10:30:00.000Z",
      "deletedAt": null,
      "user": {
        "id": "client_456def",
        "name": "John Doe",
        "email": "john.doe@example.com",
        "avatar": "https://example.com/avatars/john.jpg"
      }
    }
  ]
}
```

**المصدر:** `template/app/api/items/[slug]/comments/route.ts`

### الحصول على إحصائيات التقييم

```
GET /api/items/[slug]/comments/rating
```

إرجاع متوسط التصنيف وإجمالي عدد التصنيفات لعنصر ما. يحسب فقط التعليقات غير المحذوفة.

**رد النجاح (200):**

```json
{
  "averageRating": 4.2,
  "totalRatings": 15
}
```

إرجاع `averageRating: 0` و`totalRatings: 0` في حالة عدم وجود تقييمات.

**المصدر:** `template/app/api/items/[slug]/comments/rating/route.ts`

## نقاط النهاية المصادق عليها

### إنشاء تعليق

```
POST /api/items/[slug]/comments
```

إنشاء تعليق جديد بتقييم لعنصر ما. يتطلب المصادقة وملف تعريف العميل صالحًا. يتم منع المستخدمين المحظورين من التعليق.

**المصادقة:** مطلوب

**نص الطلب:**

```json
{
  "content": "This is an amazing tool! Really helped boost my productivity.",
  "rating": 5
}
```

|الميدان|اكتب|مطلوب|القيود|
|---|---|---|---|
|`content`|سلسلة|نعم|يجب أن تكون غير فارغة بعد التشذيب|
|`rating`|عدد صحيح|نعم|يجب أن يكون بين 1 و5 ضمناً|

**رد النجاح (200):**

```json
{
  "success": true,
  "comment": {
    "id": "comment_123abc",
    "content": "This is an amazing tool!",
    "rating": 5,
    "userId": "client_456def",
    "itemId": "item_123abc",
    "createdAt": "2024-01-20T10:30:00.000Z",
    "user": {
      "id": "client_456def",
      "name": "John Doe",
      "email": "john.doe@example.com",
      "avatar": "https://example.com/avatars/john.jpg"
    }
  }
}
```

|الحالة|الحالة|
|---|---|
| 400 |محتوى فارغ أو تقييم غير صالح|
| 401 |لم تتم المصادقة عليها|
| 403 |تم تعليق المستخدم أو حظره|
| 404 |لم يتم العثور على ملف تعريف العميل|

**المصدر:** `template/app/api/items/[slug]/comments/route.ts`

### تحديث التعليق

```
PUT /api/items/[slug]/comments/[commentId]
```

يقوم بتحديث محتوى و/أو تقييم التعليق الموجود. يمكن لمؤلف التعليق فقط تحديث التعليق الخاص به. يجب توفير واحد على الأقل من `content` أو `rating`.

**المصادقة:** مطلوب (يجب أن يكون صاحب التعليق)

**نص الطلب:**

```json
{
  "content": "Updated review text",
  "rating": 4
}
```

|الميدان|اكتب|مطلوب|القيود|
|---|---|---|---|
|`content`|سلسلة|لا|1-1000 حرف|
|`rating`|عدد صحيح|لا|يجب أن يكون بين 1 و5|

تتضمن الاستجابة التعليق المحدث مع الطابع الزمني `editedAt`.

|الحالة|الحالة|
|---|---|
| 400 |لم يتم توفير أي حقول، أو المحتوى طويل جدًا، أو التقييم غير صالح|
| 401 |لم تتم المصادقة عليها|
| 404 |لم يتم العثور على التعليق أو أن المستخدم ليس المؤلف|

**المصدر:** `template/app/api/items/[slug]/comments/[commentId]/route.ts`

### حذف التعليق

```
DELETE /api/items/[slug]/comments/[commentId]
```

يقوم بحذف التعليق بشكل بسيط. يمكن لمؤلف التعليق فقط حذف تعليقه. تم وضع علامة على التعليق بالطابع الزمني `deletedAt` بدلاً من إزالته نهائيًا.

**المصادقة:** مطلوب (يجب أن يكون صاحب التعليق)

**استجابة النجاح:** 204 لا يوجد محتوى

|الحالة|الحالة|
|---|---|
| 401 |لم تتم المصادقة عليها|
| 404 |لم يتم العثور على التعليق، أو تم حذفه بالفعل، أو أنه لا يملكه المستخدم|

**المصدر:** `template/app/api/items/[slug]/comments/[commentId]/route.ts`

### تحديث تقييم التعليق

```
PATCH /api/items/[slug]/comments/rating/[commentId]
```

يقوم بتحديث تقييم تعليق محدد فقط.

**نص الطلب:**

```json
{
  "rating": 4
}
```

**المصدر:** `template/app/api/items/[slug]/comments/rating/[commentId]/route.ts`

## نقاط نهاية المشرف

تتطلب جميع نقاط النهاية الإدارية أن يكون `session.user.isAdmin` صحيحًا.

### قائمة كافة التعليقات

```
GET /api/admin/comments
```

إرجاع قائمة مرقّمة بجميع التعليقات (باستثناء التعليقات المحذوفة مؤقتًا) مع معلومات المستخدم. يدعم البحث عبر محتوى التعليق واسم المستخدم والبريد الإلكتروني للمستخدم.

**معلمات الاستعلام:**

|المعلمة|اكتب|الافتراضي|الوصف|
|---|---|---|---|
|`page`|عدد صحيح| 1 |رقم الصفحة|
|`limit`|عدد صحيح| 10 |النتائج لكل صفحة (1-100)|
|`search`|سلسلة| - |البحث في المحتوى أو اسم المستخدم أو البريد الإلكتروني|

**رد النجاح (200):**

```json
{
  "success": true,
  "data": {
    "comments": [
      {
        "id": "comment_123abc",
        "content": "Great product!",
        "rating": 5,
        "userId": "user_456def",
        "itemId": "item_789ghi",
        "createdAt": "2024-01-20T10:30:00.000Z",
        "updatedAt": "2024-01-20T10:30:00.000Z",
        "user": {
          "id": "user_456def",
          "name": "John Doe",
          "email": "john.doe@example.com",
          "image": "https://example.com/avatar.jpg"
        }
      }
    ],
    "pagination": {
      "total": 156,
      "page": 1,
      "limit": 10,
      "totalPages": 16
    }
  }
}
```

**المصدر:** `template/app/api/admin/comments/route.ts`

### الحصول على تعليق بواسطة معرف

```
GET /api/admin/comments/[id]
```

استرداد تعليق محدد مع معلومات المستخدم الكاملة.

**المصدر:** `template/app/api/admin/comments/[id]/route.ts`

### تعليق تحديث المشرف

```
PUT /api/admin/comments/[id]
```

يسمح للمسؤولين بتحديث محتوى أي تعليق، بغض النظر عن الملكية.

**نص الطلب:**

```json
{
  "content": "This content has been moderated by an administrator."
}
```

**المصدر:** `template/app/api/admin/comments/[id]/route.ts`

### المشرف حذف التعليق

```
DELETE /api/admin/comments/[id]
```

يقوم بحذف أي تعليق بشكل بسيط. يجب أن يكون التعليق موجودًا ولم يتم حذفه بالفعل.

**رد النجاح (200):**

```json
{
  "success": true,
  "message": "Comment deleted successfully"
}
```

|الحالة|الحالة|
|---|---|
| 403 |ليس المشرف|
| 404 |لم يتم العثور على التعليق أو تم حذفه بالفعل|

**المصدر:** `template/app/api/admin/comments/[id]/route.ts`

## تفاصيل التنفيذ الرئيسية

- **الحذف البسيط:** يتم تعيين كافة عمليات الحذف على `deletedAt` بدلاً من إزالة السجلات. تقوم الاستعلامات بتصفية التعليقات المحذوفة عبر `isNull(comments.deletedAt)`.
- **التحقق من الملكية:** تتحقق نقاط نهاية المستخدم من تطابق معرف ملف تعريف العميل الخاص بالمستخدم الذي تمت مصادقته مع حقل التعليق `userId`.
- **منع المستخدم المحظور:** يمنع فحص `isUserBlocked()` المستخدمين المعلقين أو المحظورين من إنشاء التعليقات.
- **البحث (المسؤول):** يستخدم ILIKE للبحث غير الحساس لحالة الأحرف مع الهروب المناسب من أحرف البدل SQL (`%` و`_`).
