# كومة التكنولوجيا

يقدم هذا المستند نظرة عامة شاملة على جميع التقنيات المستخدمة في Ever Works.

## متطلبات النظام

- **Node.js**: 20.19.0 أو أعلى
- **PostgreSQL**: 14.0 أو أعلى
- **مدير الحزم**: npm، أو pnpm، أو الغزل، أو الكعكة

## تقنيات الواجهة الأمامية {#frontend}

### الإطار الأساسي

- **[Next.js 15.4.7](https://nextjs.org/)** - إطار عمل React باستخدام جهاز توجيه التطبيق
  - العرض من جانب الخادم (SSR)
  - إنشاء موقع ثابت (SSG)
  - التجديد الثابت التزايدي (ISR)
  - إجراءات الخادم للطفرات
  - التحسين المدمج
  - التوجيه المستند إلى الملف مع `[locale]` المقاطع الديناميكية

- **[رد فعل 19.1.0](https://react.dev/)** - مكتبة واجهة المستخدم
  - أحدث الميزات والتحسينات
  - التقديم المتزامن
  - الخلط التلقائي
  - التشويق لجلب البيانات
  - مكونات الخادم بشكل افتراضي

### سلامة اللغة والنوع

- **[TypeScript 5.x](https://www.typescriptlang.org/)** - التحقق من النوع الثابت
  - تم تمكين الوضع الصارم
  - تم تكوين تعيين المسار (`@/` الاسم المستعار)
  - تعريفات النوع المخصص
  - استنتاج النوع الكامل

### التصميم وواجهة المستخدم

- **[Tailwind CSS 3.4](https://tailwindcss.com/)** - إطار عمل CSS للأداة المساعدة أولاً
  - نظام التصميم المخصص
  - دعم الوضع المظلم
  - أدوات التصميم المستجيبة
  - تجميع JIT
  - نظام الألوان الديناميكي (50-950 درجة)

- **[HeroUI React 2.6](https://www.heroui.com/)** - مكونات React الحديثة
  - مكونات يمكن الوصول إليها
  - مواضيع قابلة للتخصيص
  - دعم تايب سكريبت
  - شجرة قابلة للاهتزاز

- **[Radix UI](https://www.radix-ui.com/)** - مكونات غير قابلة للوصول
  - البدائيات واجهة المستخدم مقطوعة الرأس
  - التنقل الكامل بلوحة المفاتيح
  - متوافق مع ARIA
  - قابل للتركيب

- **[Framer Motion 12.x](https://www.framer.com/motion/)** - مكتبة الرسوم المتحركة
  - الرسوم المتحركة التصريحية
  - دعم الإيماءات
  - الرسوم المتحركة التخطيط
  - الرسوم المتحركة SVG

### تحرير النص الغني

- **[TipTap](https://tiptap.dev/)** - محرر نص منسق بدون رأس
  - العمارة الموسعة
  - دعم تخفيض السعر
  - التحرير التعاوني جاهز
  - ملحقات مخصصة

### إدارة الدولة

- **[Zustand 5](https://zustand-demo.pmnd.rs/)** - إدارة الحالة خفيفة الوزن
  - واجهة برمجة التطبيقات البسيطة
  - دعم تايب سكريبت
  - الحد الأدنى من القالب
  - تكامل أدوات التطوير
  - دعم الوسيطة

- **[TanStack React Query 5](https://tanstack.com/query/)** - إدارة حالة الخادم
  - التخزين المؤقت والمزامنة
  - تحديثات الخلفية
  - تحديثات متفائلة
  - معالجة الأخطاء
  - استفسارات لا نهاية لها

### تصور البيانات

- **[جدول TanStack](https://tanstack.com/table/)** - مكتبة جدول بدون رأس
  - الفرز والتصفية وترقيم الصفحات
  - تغيير حجم العمود
  - اختيار الصف
  - دعم تايب سكريبت

- **[TanStack Virtual](https://tanstack.com/virtual/)** - مكتبة المحاكاة الافتراضية
  - التمرير الظاهري
  - تحسين الأداء
  - ارتفاعات الصف الديناميكية

### التعامل مع النموذج

- **[نموذج خطاف التفاعل 7](https://react-hook-form.com/)** - نماذج الأداء
  - الحد الأدنى من إعادة العرض
  - التحقق من الصحة المدمج
  - دعم تايب سكريبت
  - التكامل السهل
  - دعم المصفوفات الميدانية

- **[Zod 4](https://zod.dev/)** - التحقق من صحة المخطط
  - TypeScript-أولاً
  - التحقق من صحة وقت التشغيل
  - اكتب الاستدلال
  - معالجة الأخطاء
  - أدوات التحقق المخصصة

## تقنيات الخلفية

### قاعدة البيانات وORM

- **[PostgreSQL 14+](https://www.postgresql.org/)** - قاعدة بيانات علائقية
  - الامتثال للحمض
  - الميزات المتقدمة (JSONB، البحث عن النص الكامل)
  - أداء ممتاز
  - دعم جسون
  - المشغلات والإجراءات المخزنة

- **[Drizzle ORM 0.40.0](https://orm.drizzle.team/)** - TypeScript ORM
  - استعلامات النوع الآمن
  - الحد الأدنى من النفقات العامة
  - بناء جملة يشبه SQL
  - نظام الهجرة
  - استعلامات العلاقة
  - تصريحات معدة

- **[Supabase](https://supabase.com/)** - الواجهة الخلفية كخدمة (اختياري)
  - استضافت PostgreSQL
  - الاشتراكات في الوقت الحقيقي
  - الأمان على مستوى الصف
  - مصادقة مدمجة
  - دلاء التخزين
  - وظائف الحافة

### المصادقة

- **[NextAuth.js 5.0 (بيتا)](https://authjs.dev/)** - مكتبة المصادقة
  - موفرو OAuth متعددون (Google وGitHub وFacebook وTwitter)
  - JWT وجلسات قاعدة البيانات
  - دعم تايب سكريبت
  - أفضل الممارسات الأمنية
  - المصادقة المستندة إلى بيانات الاعتماد
  - إدارة الجلسة

- **[Supabase Auth](https://supabase.com/auth)** - حل المصادقة البديل
  - إدارة المستخدم المضمنة
  - مقدمي الخدمات الاجتماعية
  - التحقق من البريد الإلكتروني
  - إعادة تعيين كلمة المرور
  - الروابط السحرية
  - مصادقة الهاتف

### بنية المصادقة المزدوجة

يدعم Ever Works ** كلاً من NextAuth.js وSupabase Auth** في وقت واحد:

- NextAuth لتدفقات OAuth التقليدية
- Supabase Auth للميزات في الوقت الحقيقي
- إدارة الجلسة الموحدة
- التبديل السلس للمزود
