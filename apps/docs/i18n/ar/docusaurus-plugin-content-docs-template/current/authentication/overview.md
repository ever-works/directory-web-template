# نظرة عامة على المصادقة

يوفر Ever Works نظام مصادقة مرناً وآمناً يدعم موفري خدمة وأساليب مصادقة متعددة.

## بنية المصادقة

يستخدم القالب نهجاً هجيناً للمصادقة، يدعم كلاً من NextAuth.js وSupabase Auth في آنٍ واحد، مما يتيح لك اختيار الحل الأنسب لاحتياجاتك.

(mermaid diagram - keep as-is)

## أساليب المصادقة المدعومة

### 1. موفرو OAuth

NextAuth.js OAuth يدعم: Google، GitHub، Facebook، Twitter/X، Microsoft
Supabase OAuth يدعم: Google، GitHub، Facebook، Twitter/X، Discord، Apple

### 2. المصادقة بالبريد الإلكتروني/كلمة المرور

NextAuth.js Credentials: بريد إلكتروني/كلمة مرور مخصصة، تجزئة bcrypt، تخزين الجلسة في قاعدة البيانات
Supabase Auth: بريد إلكتروني/كلمة مرور مدمجة، التحقق من البريد الإلكتروني، إعادة تعيين كلمة المرور

### 3. روابط الوصول السحرية
Supabase Auth: مصادقة بدون كلمة مرور عبر روابط سحرية بالبريد الإلكتروني

### 4. WebAuthn / مفاتيح المرور
NextAuth.js: المصادقة البيومترية، مفاتيح الأمان المادية، FIDO2

## إدارة الجلسات
رموز JWT للمصادقة عديمة الحالة، جلسات قاعدة البيانات للحالة الدائمة، معالجة آمنة لملفات تعريف الارتباط، تحديث تلقائي للرموز

## ميزات الأمان
حماية CSRF، تحديد المعدل، الحماية من هجمات القوة الغاشمة، تجزئة آمنة لكلمات المرور باستخدام bcrypt
