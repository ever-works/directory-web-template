# التحكم في الوصول المستند إلى الأدوار

## نظرة عامة

يطبّق القالب نظام RBAC مع تخزين الأدوار في قاعدة البيانات.

## الأدوار الافتراضية

| الدور | الوصف |
|-------|-------|
| admin | وصول كامل للنظام |
| moderator | صلاحيات إدارة المحتوى |
| user | وصول المستخدم المصادق عليه |
| guest | وصول عام محدود |

## تعيين الأدوار

تُعيَّن الأدوار في قاعدة البيانات. يمكن للمسؤولين إدارة الأدوار من خلال لوحة التحكم على `/admin/users`.

## التحقق من الصلاحيات

```typescript
// في مسارات API
const session = await auth();
if (!session?.user?.role || session.user.role !== 'admin') {
  return NextResponse.json({ error: 'Forbidden' }, { status: 403 });
}
```

## حماية المسارات

استخدم Middleware لحماية المسارات بناءً على الأدوار. يتحقق auth middleware من الجلسة والدور قبل السماح بالوصول.
