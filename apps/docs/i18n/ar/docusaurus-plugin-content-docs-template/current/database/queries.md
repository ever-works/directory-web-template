# مرجع استعلامات قاعدة البيانات

يحتوي الدليل `lib/db/queries/` على أكثر من 23 وحدة استعلام منظمة حسب المجال. تحتوي كل وحدة على استعلامات Drizzle ORM لمنطقة ميزات محددة، وفقًا لمبدأ المسؤولية الفردية.

## نظرة عامة على الوحدة

يتم تصدير جميع وحدات الاستعلام بشكل برميلي من `lib/db/queries/index.ts` للاستيراد المريح:

```typescript
import { getUser, getUserByEmail } from '@/lib/db/queries';
```

## أنماط الاستعلام

### نمط الاستعلام القياسي

تتبع جميع وحدات الاستعلام نمطًا ثابتًا:

```typescript
import { db } from '../drizzle';
import { eq, desc, and, sql } from 'drizzle-orm';
import { tableName } from '../schema';

export async function getItemById(id: string) {
  const result = await db
    .select()
    .from(tableName)
    .where(eq(tableName.id, id))
    .limit(1);
  return result[0] || null;
}
```

### الاستعلامات المرقّمة

تقوم العديد من الوحدات بتنفيذ الاستعلامات المرقّمة:

```typescript
export async function getItems(page: number, limit: number) {
  const offset = (page - 1) * limit;
  const [items, countResult] = await Promise.all([
    db.select().from(tableName)
      .orderBy(desc(tableName.createdAt))
      .limit(limit)
      .offset(offset),
    db.select({ count: sql<number>`count(*)` })
      .from(tableName),
  ]);
  return {
    items,
    total: Number(countResult[0].count),
    page,
    limit,
  };
}
```

### استعلامات التجميع

تستخدم وحدات المشاركة ولوحة المعلومات تجميع SQL:

```typescript
export async function getEngagementScore(itemId: string) {
  const result = await db.execute(sql`
    SELECT
      COALESCE(v.vote_count, 0) as votes,
      COALESCE(c.comment_count, 0) as comments,
      COALESCE(f.favorite_count, 0) as favorites,
      COALESCE(iv.view_count, 0) as views
    FROM ...
  `);
  return result;
}
```
