# الأسئلة الشائعة

## ما هو Directory Web Template؟

Directory Web Template هو حل جاهز للإنتاج لموقع الدليل مبني على Next.js، مصمم لإطلاق أدلة الأدوات والخدمات أو المنتجات.

## هل يمكنني استخدام Template بدون Ever Works Platform؟

نعم، يعمل القالب بشكل مستقل تماماً عن منصة Ever Works.

## ما هو المكدس التقني؟

- **Next.js 15** – إطار React
- **React 19** – مكتبة UI
- **TypeScript 5** – برمجة مكتوبة
- **Tailwind CSS** – إطار CSS
- **Drizzle ORM** – إدارة قاعدة البيانات
- **NextAuth.js** – المصادقة

## ما هي مزودو OAuth المدعومون؟

- Google
- GitHub
- Facebook
- Twitter
- Microsoft

## ما هي مزودو الدفع المدعومون؟

- Stripe
- LemonSqueezy
- Polar

## ما هي منصة النشر الموصى بها؟

نوصي بـ **Vercel** للنشر الأسهل والأسرع.

## ما هي قاعدة البيانات الموصى بها؟

يُوصى باستخدام **PostgreSQL** عبر Supabase لبيئة الإنتاج.

## كيف يعمل نظام إدارة المحتوى؟

يُدار المحتوى عبر ملفات YAML/Markdown في مستودع Git المستنسخ إلى دليل `.content/`.
