# البدء مع القالب

مرحباً بك في **Directory Web Template**! سيرشدك هذا الدليل من استنساخ المستودع إلى تشغيل التطبيق.

## ما هو Ever Works Template؟

Ever Works Template هو دليل Next.js متكامل الميزات، منظم في Turborepo Monorepo. يوفر كل ما تحتاجه لإطلاق دليل الأدوات أو الخدمات أو المنتجات.

## المتطلبات

- **Node.js >= 20.19.0**
- **pnpm** – مدير الحزم
- **Next.js 16** – إطار React
- **Drizzle ORM** – إدارة قاعدة البيانات
