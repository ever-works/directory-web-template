# المرجع السريع

## الأوامر الأساسية

| الأمر | الوصف |
|-------|-------|
| `pnpm run dev` | تشغيل خادم التطوير |
| `pnpm run build` | بناء المشروع للإنتاج |
| `pnpm run lint` | فحص الكود بحثاً عن الأخطاء |
| `pnpm run db:generate` | توليد ترحيلات قاعدة البيانات |
| `pnpm run db:migrate` | تطبيق الترحيلات |
| `pnpm run db:seed` | تحميل البيانات الأولية |

## الملفات والمجلدات الرئيسية

| الملف/المجلد | الوصف |
|-------------|-------|
| `.env.local` | متغيرات البيئة المحلية |
| `.content/` | المحتوى المستنسخ من Git CMS |
| `app/` | مسارات Next.js App Router |
| `lib/` | منطق الأعمال والخدمات |
| `components/` | مكونات React |
