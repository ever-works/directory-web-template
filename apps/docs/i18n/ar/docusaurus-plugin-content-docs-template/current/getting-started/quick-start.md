# البداية السريعة

## الخطوة الأولى: تكوين config.yml

قم بتحرير ملف `config.yml` لضبط المعاملات الأساسية للدليل الخاص بك.

## الخطوة الثانية: Fork مستودع البيانات

1. قم بعمل Fork لمستودع البيانات على GitHub
2. قم بتعيين متغيرات البيئة التالية:
   - `DATA_REPOSITORY` – رابط مستودعك المتفرع
   - `GH_TOKEN` – رمز وصول GitHub الخاص بك

## الخطوة الثالثة: تشغيل خادم التطوير

```bash
pnpm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في متصفحك.

## الخطوة الرابعة: إضافة أول عنصر

أنشئ ملف YAML جديداً في دليل البيانات:

```yaml
name: "أداتي"
description: "وصف الأداة"
url: "https://example.com"
category: "أدوات"
```
