# Directory Web Template

**Directory Web Template** هو حل متكامل وعصري لموقع الدليل، مبني باستخدام Next.js 16. مصمم لمواقع الدليل الاحترافية للأدوات والخدمات والمنتجات أو منصات القوائم.

## الميزات الرئيسية

- **مكدس تقني حديث** – Next.js 16 وReact 19 وTypeScript
- **Turborepo Monorepo** – هيكل منظم للمشروع
- **مصادقة مرنة** – NextAuth.js v5 ومزودو OAuth
- **تكامل الدفع** – Stripe وLemonSqueezy وPolar
- **التدويل** – دعم متعدد اللغات
- **نظام إدارة المحتوى المستند إلى Git** – إدارة المحتوى عبر Git
- **نظام السمات** – تصميم مخصص
- **التحليلات والمراقبة** – تحليلات مدمجة
- **لوحة الإدارة** – إدارة المحتوى
- **تحسين محركات البحث** – محسّن لمحركات البحث

## البداية السريعة

```bash
git clone https://github.com/ever-works/directory-web-template.git
cd directory-web-template
pnpm install
pnpm run dev
```
