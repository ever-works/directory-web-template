# دليل تدريب الفريق

برنامج تدريبي شامل لإلحاق أعضاء الفريق وتطوير مهاراتهم في منصة Ever Works.

## 🎯 أهداف التدريب

بعد إتمام البرنامج التدريبي، سيتمكن أعضاء الفريق من:

- ✅ إعداد بيئة تطوير العمل
- ✅ فهم بنية Ever Works ومكدس التقنيات
- ✅ اتباع معايير الكود والاتفاقيات المعتمدة
- ✅ توثيق API باستخدام نظام التوثيق الآلي
- ✅ تنفيذ الميزات وفق أفضل الممارسات
- ✅ تشخيص المشكلات الشائعة وحلها
- ✅ المساهمة في قاعدة الكود بثقة

## 📚 وحدات التدريب

### 1. [دليل الإلحاق](/team-training/onboarding)

**المدة**: يوم إلى يومين  
**الجمهور**: الأعضاء الجدد في الفريق

### 2. [توثيق API](/team-training/api-documentation)

**المدة**: يومان إلى ثلاثة أيام  
**الجمهور**: جميع المطورين العاملين مع نقاط نهاية API

### 3. [أفضل الممارسات](/team-training/best-practices)

**المدة**: يوم إلى يومين  
**الجمهور**: جميع أعضاء الفريق

### 4. [التمارين العملية](/team-training/exercises)

**المدة**: ثلاثة إلى خمسة أيام  
**الجمهور**: جميع أعضاء الفريق

## 📅 جدول التدريب

### الأعضاء الجدد (أسبوعان)

**الأسبوع 1: الأساسيات**
- الأيام 1-2: [دليل الإلحاق](/team-training/onboarding)

**الأسبوع 2: التطبيق العملي**

## 📋 المتطلبات المسبقة

### المعرفة المطلوبة

- **JavaScript/TypeScript**: متوسط فأعلى
- **React**: فهم جيد للـ Hooks والمكوّنات وإدارة الحالة
- **Next.js**: إلمام أساسي (يُفضَّل App Router)
- **Git**: أساسيات التحكم في الإصدارات

### الأدوات المطلوب تثبيتها

- Node.js 20.19.0+ (استخدم nvm)
- pnpm ([التثبيت](https://pnpm.io/installation))
- PostgreSQL 14+
- Git
- VS Code (أو البيئة المفضلة)

## 🆘 الحصول على المساعدة

**أثناء التدريب**:
- استشر المرشد أو قائد الفريق التقني
- راجع [دليل المرجع السريع](/getting-started/quick-reference)

**بعد التدريب**:
- قنوات Slack الخاصة بالفريق
- ملاحظات مراجعة الكود
- اجتماعات الفريق الأسبوعية
