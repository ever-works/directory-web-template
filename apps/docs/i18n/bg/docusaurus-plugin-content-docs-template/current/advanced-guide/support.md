## 🔧 Отстраняване на неизправности

### Често срещани проблеми

#### Проблеми с инсталацията

- **Node.js версия**: Уверете се, че използвате Node.js 18+
- **Зависимости**: Стартирайте `npm install` или `yarn install` в директорията на уебсайта
- **Конфликти на портове**: Използвайте флаг `--port` , за да посочите различен порт

#### Проблеми с изграждането

- **Грешки на TypeScript**: Проверете вашата конфигурация на TypeScript
- **Липсващи зависимости**: Уверете се, че всички пакети са правилно инсталирани
- **Променливи на средата**: Проверете вашата конфигурация на файла `.env` #### Проблеми по време на работа

- **API връзка**: Проверете вашата променлива на средата `NEXT_PUBLIC_API_BASE_URL` - **Удостоверяване**: Проверете идентификационните данни на доставчика на OAuth
- **База данни**: Уверете се, че връзката ви с база данни работи

### Режим на отстраняване на грешки

Активирайте регистрирането на грешки чрез задаване на променливи на средата:

```bash
DEBUG=directory-web-template:*
NODE_ENV=development
```
