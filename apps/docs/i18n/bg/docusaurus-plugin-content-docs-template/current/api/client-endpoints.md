# Крайни точки на API на клиента

Крайните точки на API, насочени към клиента, обслужват удостоверени крайни потребители (без администратори). Тези маршрути обработват таблото за управление на клиента, подаването на елементи, управлението на любимите и публичните взаимодействия с елементи, като коментари, гласове и изгледи.

## Клиентско табло и елементи (`/api/client`)

Всички `/api/client/*` маршрути изискват удостоверена сесия с валиден `clientProfileId`.

### Табло за управление

|Метод|Пътека|Описание|
|--------|------|-------------|
|`GET`|`/api/client/dashboard/stats`|Статистика на таблото за управление на клиента (брой елементи, показвания, ангажираност)|

### Клиентски артикули

|Метод|Пътека|Описание|
|--------|------|-------------|
|`GET`|`/api/client/items`|Избройте елементи, изпратени от текущия клиент|
|`POST`|`/api/client/items`|Изпратете нов артикул за преглед|
|`GET`|`/api/client/items/stats`|Статистика на клиентски артикул (публикувани, чакащи, отхвърлени)|
|`GET`|`/api/client/items/coordinates`|Вземете координати за артикули на клиента|
|`GET`|`/api/client/items/[id]`|Получете подробности за артикула|
|`PUT`|`/api/client/items/[id]`|Актуализиране на собствен артикул|
|`DELETE`|`/api/client/items/[id]`|Изтриване на собствен елемент (плавно изтриване)|
|`POST`|`/api/client/items/[id]/restore`|Възстановете меко изтрит елемент|

### Географски статистики

|Метод|Пътека|Описание|
|--------|------|-------------|
|`GET`|`/api/client/geo-stats`|Географска статистика за клиентски артикули|

## Взаимодействия с обществени елементи (`/api/items`)

Тези крайни точки обработват функциите на публичните елементи. Някои изискват удостоверяване (напр. гласуване), докато други са напълно публични (напр. гледане).

### Коментари

|Метод|Пътека|Описание|авт|
|--------|------|-------------|------|
|`GET`|`/api/items/[slug]/comments`|Избройте коментари за даден елемент|Обществен|
|`POST`|`/api/items/[slug]/comments`|Добавяне на коментар|Задължително|
|`GET`|`/api/items/[slug]/comments/[commentId]`|Получете подробности за коментара|Обществен|
|`PUT`|`/api/items/[slug]/comments/[commentId]`|Актуализирайте собствен коментар|Задължително|
|`DELETE`|`/api/items/[slug]/comments/[commentId]`|Изтриване на собствен коментар|Задължително|

### Оценки на коментари

|Метод|Пътека|Описание|авт|
|--------|------|-------------|------|
|`GET`|`/api/items/[slug]/comments/rating`|Вземете обобщена оценка|Обществен|
|`POST`|`/api/items/[slug]/comments/rating`|Изпратете оценка|Задължително|
|`GET`|`/api/items/[slug]/comments/rating/[commentId]`|Вземете оценка за коментар|Обществен|

### Гласове

|Метод|Пътека|Описание|авт|
|--------|------|-------------|------|
|`GET`|`/api/items/[slug]/votes/count`|Получете преброяване на гласовете|Обществен|
|`GET`|`/api/items/[slug]/votes/status`|Получете статус на гласуване на текущия потребител|Задължително|
|`POST`|`/api/items/[slug]/votes`|Гласуване за елемент (гласуване за/против)|Задължително|

### Изгледи

|Метод|Пътека|Описание|авт|
|--------|------|-------------|------|
|`POST`|`/api/items/[slug]/views`|Запишете изглед на страница|Обществен|

### Ангажираност и популярност

## Любими (`/api/favorites`)

Управлявайте любимите елементи на потребителя. Всички любими крайни точки изискват удостоверяване.

|Метод|Пътека|Описание|
|--------|------|-------------|
|`GET`|`/api/favorites`|Избройте любимите елементи на текущия потребител|
|`POST`|`/api/favorites/[itemSlug]`|Превключване на предпочитано състояние за даден елемент|
|`DELETE`|`/api/favorites/[itemSlug]`|Премахване на елемент от любими|

|`POST`|`/api/sponsor-ads/checkout`|Започнете плащане на реклама на спонсор|

## Модели за пагинация

Крайните точки на списъка, обърнат към клиента, поддържат стандартните параметри за страниране:

```
GET /api/client/items?page=1&limit=10&sort=createdAt&order=desc
GET /api/items/[slug]/comments?page=1&limit=20
GET /api/favorites?page=1&limit=50
```

Отговорите включват метаданни за страниране:

```json
{
  "success": true,
  "data": [...],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 45,
    "totalPages": 5
  }
}
```
