# Коментирайте крайни точки

Системата за коментари предоставя крайни точки за създаване, четене, актуализиране и изтриване на коментари за елементи. Коментарите включват оценка от 1 до 5 звезди и поддържат както публичен достъп (четене), така и удостоверени операции (създаване/редактиране/изтриване). Крайните точки на администратора предоставят възможности за модериране.

## Преглед

### Публични крайни точки

|Крайна точка|Метод|авт|Описание|
|---|---|---|---|
|`/api/items/[slug]/comments`|ВЗЕМЕТЕ|Обществен|Избройте коментари за даден елемент|
|`/api/items/[slug]/comments/rating`|ВЗЕМЕТЕ|Обществен|Вземете обобщена статистика за рейтинг|
|`/api/items/[slug]/comments/rating/[commentId]`|ВЗЕМЕТЕ|Обществен|Получете оценка от един коментар|

### Удостоверени крайни точки

|Крайна точка|Метод|авт|Описание|
|---|---|---|---|
|`/api/items/[slug]/comments`|ПУБЛИКУВАНЕ|Потребител|Създайте нов коментар|
|`/api/items/[slug]/comments/[commentId]`|ПОСТАВЕТЕ|Собственик|Актуализирайте собствен коментар|
|`/api/items/[slug]/comments/[commentId]`|ИЗТРИВАНЕ|Собственик|Изтриване на собствен коментар|
|`/api/items/[slug]/comments/rating/[commentId]`|КРЕПКА|Потребител|Актуализирайте оценката на коментар|

### Административни крайни точки

|Крайна точка|Метод|авт|Описание|
|---|---|---|---|
|`/api/admin/comments`|ВЗЕМЕТЕ|Админ|Избройте всички коментари със страници|
|`/api/admin/comments/[id]`|ВЗЕМЕТЕ|Админ|Вземете коментар по ID|
|`/api/admin/comments/[id]`|ПОСТАВЕТЕ|Админ|Актуализирайте съдържанието на коментара|

## Публични крайни точки

### Коментари на елемент от списъка

```
GET /api/items/[slug]/comments
```

Връща всички коментари за конкретен елемент, включително информация за потребителския профил. Не се изисква удостоверяване.

**Параметри на пътя:**

|Параметър|Тип|Описание|
|---|---|---|
|`slug`|низ|елемент плужек|

**Успешен отговор (200):**

```json
{
  "success": true,
  "comments": [
    {
      "id": "comment_123abc",
      "content": "This is an amazing tool!",
      "rating": 5,
      "userId": "client_456def",
      "itemId": "item_123abc",
      "createdAt": "2024-01-20T10:30:00.000Z",
      "updatedAt": "2024-01-20T10:30:00.000Z",
      "deletedAt": null,
      "user": {
        "id": "client_456def",
        "name": "John Doe",
        "email": "john.doe@example.com",
        "avatar": "https://example.com/avatars/john.jpg"
      }
    }
  ]
}
```

**Източник:** `template/app/api/items/[slug]/comments/route.ts`

### Вземете статистика за рейтинга

```
GET /api/items/[slug]/comments/rating
```

Връща средната оценка и общия брой оценки за артикул. Броят се само неизтритите коментари.

**Успешен отговор (200):**

```json
{
  "averageRating": 4.2,
  "totalRatings": 15
}
```

Връща `averageRating: 0` и `totalRatings: 0`, когато няма оценки.

**Източник:** `template/app/api/items/[slug]/comments/rating/route.ts`

## Удостоверени крайни точки

### Създаване на коментар

```
POST /api/items/[slug]/comments
```

Създава нов коментар с оценка на елемент. Изисква удостоверяване и валиден клиентски профил. Блокираните потребители не могат да коментират.

**Удостоверяване:** Изисква се

**Тяло на заявката:**

```json
{
  "content": "This is an amazing tool! Really helped boost my productivity.",
  "rating": 5
}
```

|Поле|Тип|Задължително|Ограничения|
|---|---|---|---|
|`content`|низ|да|Не трябва да е празно след изрязване|
|`rating`|цяло число|да|Трябва да е между 1 и 5 включително|

**Успешен отговор (200):**

```json
{
  "success": true,
  "comment": {
    "id": "comment_123abc",
    "content": "This is an amazing tool!",
    "rating": 5,
    "userId": "client_456def",
    "itemId": "item_123abc",
    "createdAt": "2024-01-20T10:30:00.000Z",
    "user": {
      "id": "client_456def",
      "name": "John Doe",
      "email": "john.doe@example.com",
      "avatar": "https://example.com/avatars/john.jpg"
    }
  }
}
```

|Статус|Състояние|
|---|---|
| 400 |Празно съдържание или невалидна оценка|
| 401 |Не е удостоверено|
| 403 |Потребителят е спрян или забранен|
| 404 |Клиентският профил не е намерен|

**Източник:** `template/app/api/items/[slug]/comments/route.ts`

### Коментар за актуализиране

```
PUT /api/items/[slug]/comments/[commentId]
```

Актуализира съдържанието и/или оценката на съществуващ коментар. Само авторът на коментара може да актуализира собствения си коментар. Трябва да бъде предоставен поне един от `content` или `rating`.

**Удостоверяване:** Изисква се (трябва да сте собственик на коментар)

**Тяло на заявката:**

```json
{
  "content": "Updated review text",
  "rating": 4
}
```

|Поле|Тип|Задължително|Ограничения|
|---|---|---|---|
|`content`|низ|не|1-1000 знака|
|`rating`|цяло число|не|Трябва да е между 1 и 5|

Отговорът включва актуализирания коментар с клеймо за време `editedAt`.

|Статус|Състояние|
|---|---|
| 400 |Няма предоставени полета, твърде дълго съдържание или невалидна оценка|
| 401 |Не е удостоверено|
| 404 |Коментарът не е намерен или потребителят не е авторът|

**Източник:** `template/app/api/items/[slug]/comments/[commentId]/route.ts`

### Изтриване на коментар

```
DELETE /api/items/[slug]/comments/[commentId]
```

Плавно изтрива коментар. Само авторът на коментара може да изтрие собствения си коментар. Коментарът е маркиран с `deletedAt` клеймо за време, вместо да бъде премахнат за постоянно.

**Удостоверяване:** Изисква се (трябва да сте собственик на коментар)

**Успешен отговор:** 204 Няма съдържание

|Статус|Състояние|
|---|---|
| 401 |Не е удостоверено|
| 404 |Коментарът не е намерен, вече е изтрит или не е собственост на потребителя|

**Източник:** `template/app/api/items/[slug]/comments/[commentId]/route.ts`

### Актуализиране на оценката на коментара

```
PATCH /api/items/[slug]/comments/rating/[commentId]
```

Актуализира само оценката на конкретен коментар.

**Тяло на заявката:**

```json
{
  "rating": 4
}
```

**Източник:** `template/app/api/items/[slug]/comments/rating/[commentId]/route.ts`

## Административни крайни точки

Всички администраторски крайни точки изискват `session.user.isAdmin` да е вярно.

### Избройте всички коментари

```
GET /api/admin/comments
```

Връща списък със страници с всички коментари (с изключение на тези, които са изтрити меко) с потребителска информация. Поддържа търсене в съдържание на коментари, потребителско име и потребителски имейл.

**Параметри на заявката:**

|Параметър|Тип|По подразбиране|Описание|
|---|---|---|---|
|`page`|цяло число| 1 |Номер на страницата|
|`limit`|цяло число| 10 |Резултати на страница (1-100)|
|`search`|низ| - |Търсете в съдържание, потребителско име или имейл|

**Успешен отговор (200):**

```json
{
  "success": true,
  "data": {
    "comments": [
      {
        "id": "comment_123abc",
        "content": "Great product!",
        "rating": 5,
        "userId": "user_456def",
        "itemId": "item_789ghi",
        "createdAt": "2024-01-20T10:30:00.000Z",
        "updatedAt": "2024-01-20T10:30:00.000Z",
        "user": {
          "id": "user_456def",
          "name": "John Doe",
          "email": "john.doe@example.com",
          "image": "https://example.com/avatar.jpg"
        }
      }
    ],
    "pagination": {
      "total": 156,
      "page": 1,
      "limit": 10,
      "totalPages": 16
    }
  }
}
```

**Източник:** `template/app/api/admin/comments/route.ts`

### Вземете коментар по ID

```
GET /api/admin/comments/[id]
```

Извлича конкретен коментар с пълна информация за потребителя.

**Източник:** `template/app/api/admin/comments/[id]/route.ts`

### Коментар на актуализацията на администратора

```
PUT /api/admin/comments/[id]
```

Позволява на администраторите да актуализират съдържанието на всеки коментар, независимо от собствеността.

**Тяло на заявката:**

```json
{
  "content": "This content has been moderated by an administrator."
}
```

**Източник:** `template/app/api/admin/comments/[id]/route.ts`

### Администратор Изтриване на коментар

```
DELETE /api/admin/comments/[id]
```

Плавно изтрива всеки коментар. Коментарът трябва да съществува и вече да не е изтрит.

**Успешен отговор (200):**

```json
{
  "success": true,
  "message": "Comment deleted successfully"
}
```

|Статус|Състояние|
|---|---|
| 403 |Не е администратор|
| 404 |Коментарът не е намерен или вече е изтрит|

**Източник:** `template/app/api/admin/comments/[id]/route.ts`

## Ключови подробности за внедряването

- **Меко изтриване:** Всички изтривания задават `deletedAt` вместо да премахват записи. Заявките филтрират изтритите коментари чрез `isNull(comments.deletedAt)`.
- **Проверка на собствеността:** Потребителските крайни точки проверяват, че идентификаторът на клиентския профил на удостоверения потребител съответства на полето `userId` на коментара.
- **Превенция на блокирани потребители:** Проверката `isUserBlocked()` не позволява на спрени или забранени потребители да създават коментари.
- **Търсене (администратор):** Използва ILIKE за търсене без разлика на главни и малки букви с правилно екраниране на SQL заместващи знаци (`%` и `_`).
