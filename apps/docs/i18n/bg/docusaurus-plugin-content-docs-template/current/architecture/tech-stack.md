# Технологичен стек

Този документ предоставя изчерпателен преглед на всички технологии, използвани в Ever Works.

## Системни изисквания

- **Node.js**: 20.19.0 или по-висока
- **PostgreSQL**: 14.0 или по-висока
- **Мениджър на пакети**: npm, pnpm, yarn или bun

## Frontend технологии {#frontend}

### Основна рамка

- **[Next.js 15.4.7](https://nextjs.org/)** - React framework с App Router
  - Изобразяване от страна на сървъра (SSR)
  - Генериране на статичен сайт (SSG)
  - Инкрементална статична регенерация (ISR)
  - Действия на сървъра за мутации
  - Вградена оптимизация
  - Файлово базирано маршрутизиране с `[locale]` динамични сегменти

- **[React 19.1.0](https://react.dev/)** - UI библиотека
  - Най-новите функции и подобрения
  - Едновременно изобразяване
  - Автоматично дозиране
  - Суспензия за извличане на данни
  - Компоненти на сървъра по подразбиране

### Безопасност на език и тип

- **[TypeScript 5.x](https://www.typescriptlang.org/)** - Статична проверка на типа
  - Активиран строг режим
  - Конфигурирано съпоставяне на пътя (`@/` псевдоним)
  - Персонализирани дефиниции на типове
  - Извод за пълен тип

### Стил и потребителски интерфейс

- **[Tailwind CSS 3.4](https://tailwindcss.com/)** - Първата CSS рамка за помощни програми
  - Система за проектиране по поръчка
  - Поддръжка на тъмен режим
  - Помощни програми за адаптивен дизайн
  - JIT компилация
  - Динамична цветова система (50-950 нюанса)

- **[HeroUI React 2.6](https://www.heroui.com/)** - Modern React компоненти
  - Достъпни компоненти
  - Персонализируеми теми
  - Поддръжка на TypeScript
  - Дърво-клатещо се

- **[Radix UI](https://www.radix-ui.com/)** - Нестилизирани достъпни компоненти
  - Примитиви на потребителския интерфейс без глава
  - Пълна навигация с клавиатура
  - Съвместим с ARIA
  - Съставим

- **[Framer Motion 12.x](https://www.framer.com/motion/)** - Библиотека с анимации
  - Декларативни анимации
  - Поддръжка на жестове
  - Анимации на оформлението
  - SVG анимации

### Редактиране на богат текст

  - Разширяема архитектура

## Бекенд технологии

### База данни и ORM

- **[PostgreSQL 14+](https://www.postgresql.org/)** - Релационна база данни
  - Съответствие с ACID
  - Разширени функции (JSONB, търсене в пълен текст)
  - Отлично представяне
  - Поддръжка на JSON
  - Тригери и съхранени процедури

- **[Drizzle ORM 0.40.0](https://orm.drizzle.team/)** - TypeScript ORM
  - Типобезопасни заявки
  - Минимални режийни разходи
  - SQL-подобен синтаксис
  - Система за миграция
  - Релационни запитвания
  - Подготвени отчети

- **[Supabase](https://supabase.com/)** - Backend-as-a-Service (по избор)
  - Хостван PostgreSQL
  - Абонаменти в реално време
  - Сигурност на ниво ред
  - Вградена авт
  - Кофи за съхранение
  - Крайни функции

### Удостоверяване

- **[NextAuth.js 5.0 (бета)](https://authjs.dev/)** - Библиотека за удостоверяване
  - Множество доставчици на OAuth (Google, GitHub, Facebook, Twitter)
  - JWT и сесии на база данни
  - Поддръжка на TypeScript
  - Най-добри практики за сигурност
  - Удостоверяване, базирано на идентификационни данни
  - Управление на сесии

- **[Supabase Auth](https://supabase.com/auth)** - Алтернативно решение за удостоверяване
  - Вградено управление на потребителите
  - Социални доставчици
  - Проверка на имейл
  - Нулиране на парола
  - Магически връзки
  - Удостоверяване на телефона

### Архитектура с двойно удостоверяване

Ever Works поддържа **както NextAuth.js, така и Supabase Auth** едновременно:

- NextAuth за традиционни OAuth потоци
- Supabase Auth за функции в реално време
- Унифицирано управление на сесии
- Безпроблемна смяна на доставчик
