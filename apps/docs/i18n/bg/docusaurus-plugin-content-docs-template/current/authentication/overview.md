# Преглед на Удостоверяването

Ever Works предоставя гъвкава, сигурна система за удостоверяване, която поддържа множество доставчици и методи за удостоверяване.

## Архитектура на Удостоверяването

Шаблонът използва хибриден подход към удостоверяването, като поддържа едновременно NextAuth.js и Supabase Auth, позволявайки ви да изберете най-доброто решение за вашите нужди.

## Поддържани Методи за Удостоверяване

### 1. OAuth Доставчици

NextAuth.js OAuth поддържа: Google, GitHub, Facebook, Twitter/X, Microsoft
Supabase OAuth поддържа: Google, GitHub, Facebook, Twitter/X, Discord, Apple

### 2. Удостоверяване с Email/Парола

NextAuth.js Credentials: Персонализиран email/парола, bcrypt хеширане, съхранение на сесии в база данни
Supabase Auth: Вграден email/парола, верификация на email, нулиране на парола

### 3. Magic Links
Supabase Auth: Удостоверяване без парола чрез magic връзки по email

### 4. WebAuthn / Passkeys
NextAuth.js: Биометрично удостоверяване, хардуерни ключове за сигурност, FIDO2

## Управление на Сесии
JWT токени за безсъстояние удостоверяване, сесии в база данни за постоянно състояние, сигурна обработка на бисквитки, автоматично обновяване на токени

## Функции за Сигурност
Защита от CSRF, ограничаване на скоростта, защита от груба сила, сигурно хеширане на пароли с bcrypt
