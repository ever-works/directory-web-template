# Справочник за заявки към бази данни

Директорията `lib/db/queries/` съдържа 23+ модула за заявки, организирани по домейн. Всеки модул капсулира Drizzle ORM заявки за конкретна функционална област, следвайки принципа на единната отговорност.

## Преглед на модула

Всички модули за заявки се експортират от `lib/db/queries/index.ts` за удобно импортиране:

```typescript
import { getUser, getUserByEmail } from '@/lib/db/queries';
```

## Модели на заявки

### Стандартен модел на заявка

Всички модули за заявки следват последователен модел:

```typescript
import { db } from '../drizzle';
import { eq, desc, and, sql } from 'drizzle-orm';
import { tableName } from '../schema';

export async function getItemById(id: string) {
  const result = await db
    .select()
    .from(tableName)
    .where(eq(tableName.id, id))
    .limit(1);
  return result[0] || null;
}
```

### Пагинирани заявки

Много модули прилагат странирани заявки:

```typescript
export async function getItems(page: number, limit: number) {
  const offset = (page - 1) * limit;
  const [items, countResult] = await Promise.all([
    db.select().from(tableName)
      .orderBy(desc(tableName.createdAt))
      .limit(limit)
      .offset(offset),
    db.select({ count: sql<number>`count(*)` })
      .from(tableName),
  ]);
  return {
    items,
    total: Number(countResult[0].count),
    page,
    limit,
  };
}
```

### Заявки за агрегиране

Модулите за ангажираност и таблото за управление използват SQL агрегиране:

```typescript
export async function getEngagementScore(itemId: string) {
  const result = await db.execute(sql`
    SELECT
      COALESCE(v.vote_count, 0) as votes,
      COALESCE(c.comment_count, 0) as comments,
      COALESCE(f.favorite_count, 0) as favorites,
      COALESCE(iv.view_count, 0) as views
    FROM ...
  `);
  return result;
}
```
