# Често задавани въпроси

## Какво е Directory Web Template?

Directory Web Template е готово за производство решение за уебсайт-директория, изградено с Next.js, предназначено за стартиране на директории за инструменти, услуги или продукти.

## Мога ли да използвам Template без Ever Works Platform?

Да, шаблонът работи напълно независимо от платформата Ever Works.

## Какъв е технологичният стек?

- **Next.js 15** – React рамка
- **React 19** – UI библиотека
- **TypeScript 5** – типизирано програмиране
- **Tailwind CSS** – CSS рамка
- **Drizzle ORM** – управление на база данни
- **NextAuth.js** – удостоверяване

## Кои OAuth провайдъри се поддържат?

- Google
- GitHub
- Facebook
- Twitter
- Microsoft

## Кои платежни провайдъри се поддържат?

- Stripe
- LemonSqueezy
- Polar

## Каква е препоръчаната платформа за внедряване?

Препоръчваме **Vercel** за най-лесното и бързо внедряване.

## Каква база данни се препоръчва?

**PostgreSQL** чрез Supabase за производствена среда.

## Как работи CMS системата?

Съдържанието се управлява чрез YAML/Markdown файлове в Git репозитория, клониран в директория `.content/`.
