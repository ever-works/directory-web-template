# Начало с шаблона

Добре дошли в **Directory Web Template**! Това ръководство ще ви преведе от клониране на репозитория до стартиране на работещо приложение.

## Какво е Ever Works Template?

Ever Works Template е пълнофункционална Next.js директория, организирана в Turborepo монорепо. Предоставя всичко необходимо за стартиране на директория за инструменти, услуги или продукти.

## Изисквания

- **Node.js >= 20.19.0**
- **pnpm** – мениджър на пакети
- **Next.js 16** – React рамка
- **Drizzle ORM** – управление на база данни
