# Бърза справка

## Основни команди

| Команда | Описание |
|---------|----------|
| `pnpm run dev` | Стартира сървъра за разработка |
| `pnpm run build` | Изгражда проекта за производство |
| `pnpm run lint` | Проверява кода за грешки |
| `pnpm run db:generate` | Генерира миграции на базата данни |
| `pnpm run db:migrate` | Прилага миграциите |
| `pnpm run db:seed` | Зарежда началните данни |

## Ключови файлове и директории

| Файл/Директория | Описание |
|-----------------|----------|
| `.env.local` | Локални средови променливи |
| `.content/` | Клонирано съдържание от Git CMS |
| `app/` | Next.js App Router маршрути |
| `lib/` | Бизнес логика и услуги |
| `components/` | React компоненти |
