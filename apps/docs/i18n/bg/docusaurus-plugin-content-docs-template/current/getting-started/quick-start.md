# Бърз старт

## Стъпка 1: Конфигурирайте config.yml

Редактирайте файла `config.yml` за настройване на основните параметри на вашата директория.

## Стъпка 2: Разклонете репозитория с данни

1. Разклонете репозитория с данни в GitHub
2. Задайте следните средови променливи:
   - `DATA_REPOSITORY` – URL на вашия разклонен репозиторий
   - `GH_TOKEN` – вашият GitHub токен за достъп

## Стъпка 3: Стартирайте сървъра за разработка

```bash
pnpm run dev
```

Отворете [http://localhost:3000](http://localhost:3000) в браузъра си.

## Стъпка 4: Добавете първия си елемент

Създайте нов YAML файл в директорията с данни:

```yaml
name: "Моят инструмент"
description: "Описание на инструмента"
url: "https://example.com"
category: "инструменти"
```
