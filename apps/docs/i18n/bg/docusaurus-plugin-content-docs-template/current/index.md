# Directory Web Template

**Directory Web Template** е модерно, full-stack решение за уебсайт-директория, изградено с Next.js 16. Проектирано за професионални уебсайт-директории за инструменти, услуги, продукти или платформи за обяви.

## Основни функции

- **Модерен технологичен стек** – Next.js 16, React 19, TypeScript
- **Turborepo монорепо** – организирана структура на проекта
- **Гъвкаво удостоверяване** – NextAuth.js v5, OAuth провайдъри
- **Интеграция на плащания** – Stripe, LemonSqueezy, Polar
- **Интернационализация** – многоезична поддръжка
- **CMS базиран на Git** – управление на съдържание чрез Git
- **Система за теми** – персонализиран дизайн
- **Анализи и наблюдение** – вградена аналитика
- **Административно табло** – управление на съдържание
- **SEO оптимизация** – оптимизирано за търсачки

## Бърз старт

```bash
git clone https://github.com/ever-works/directory-web-template.git
cd directory-web-template
pnpm install
pnpm run dev
```
