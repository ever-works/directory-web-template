# Ръководство за Обучение на Екипа

Цялостна програма за обучение за въвеждане в работа и повишаване на квалификацията на екипи от разработчици на платформата Ever Works.

## 🎯 Цели на обучението

След завършване на програмата членовете на екипа ще могат да:

- ✅ Настройват работна среда за разработка
- ✅ Разбират архитектурата и технологичния стек на Ever Works
- ✅ Следват установените стандарти и конвенции за код
- ✅ Документират API с автоматизираната система за документация
- ✅ Реализират функции според най-добрите практики
- ✅ Отстраняват и решават типични проблеми
- ✅ Уверено допринасят за кодовата база

## 📚 Учебни модули

### 1. [Въвеждане в работа](/team-training/onboarding)

**Продължителност**: 1–2 дни  
**Аудитория**: Нови членове на екипа

### 2. [API Документация](/team-training/api-documentation)

**Продължителност**: 2–3 дни  
**Аудитория**: Всички разработчици, работещи с API крайни точки

### 3. [Най-добри практики](/team-training/best-practices)

**Продължителност**: 1–2 дни  
**Аудитория**: Всички членове на екипа

### 4. [Практически упражнения](/team-training/exercises)

**Продължителност**: 3–5 дни  
**Аудитория**: Всички членове на екипа

## 📅 График на обучението

### Нови членове на екипа (2 седмици)

**Седмица 1: Основи**
- Дни 1–2: [Въвеждане в работа](/team-training/onboarding)
- Дни 3–4: [API Документация](/team-training/api-documentation)
- Ден 5: [Най-добри практики](/team-training/best-practices)

**Седмица 2: Практика**
- Дни 1–3: [Практически упражнения](/team-training/exercises)
- Дни 4–5: Първа реална задача с ментор

## 📋 Предварителни изисквания

### Необходими знания

- **JavaScript/TypeScript**: Средно ниво и нагоре
- **React**: Добро разбиране на hooks, компоненти, управление на състоянието
- **Next.js**: Основно запознаване (предпочита се App Router)
- **Git**: Основи на контрол на версии

### Инструменти за инсталиране

- Node.js 20.19.0+ (използвайте nvm)
- pnpm ([инсталиране](https://pnpm.io/installation))
- PostgreSQL 14+
- Git
- VS Code (или предпочитана IDE)

## 🆘 Получаване на помощ

**По време на обучението**:
- Попитайте ментора или тийм лийда
- Вижте [Ръководство за бърз достъп](/getting-started/quick-reference)

**След обучението**:
- Slack канали на екипа
- Обратна връзка при преглед на код
- Седмични срещи на екипа
