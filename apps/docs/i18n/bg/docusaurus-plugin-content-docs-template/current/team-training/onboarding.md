# Ръководство за Включване

Добре дошли в Ever Works! Това ръководство ще ви помогне да настроите работната среда и да направите първия си принос.

## 🎯 Цели

След завършване на този модул:

- ✅ Ще имате напълно настроена среда за разработка
- ✅ Ще разберете структурата на проекта
- ✅ Ще можете да стартирате приложението локално
- ✅ Ще направите първата промяна в кода
- ✅ Ще разберете работния процес за разработка

**Приблизително време**: 1–2 дни

---

## Стъпка 1: Настройка на средата

### 1.1 Инсталирайте необходимите инструменти

Следвайте подробното [Ръководство за инсталиране](/getting-started/installation) за инсталиране на:

- Node.js 20.19.0+
- pnpm ([инсталиране](https://pnpm.io/installation))
- PostgreSQL 14+
- Git
- VS Code (препоръчан)

### 1.2 Клонирайте хранилището

```bash
git clone https://github.com/ever-co/ever-works.git
cd ever-works
pnpm install
```

### 1.3 Конфигурирайте променливите на средата

**Бърз контролен списък**:

- [ ] Връзката с базата данни е конфигурирана
- [ ] Тайните за удостоверяване са зададени
- [ ] Ключовете за платежния доставчик са добавени (незадължително за разработка)

---

## Стъпка 2: Настройка на базата данни

### 2.1 Стартирайте PostgreSQL

```bash
docker run --name everworks-postgres \
  -e POSTGRES_PASSWORD=password \
  -e POSTGRES_DB=everworks \
  -p 5432:5432 \
  -d postgres:14
```

### 2.2 Изпълнете миграциите

```bash
cd apps/web
pnpm exec drizzle-kit push
pnpm run db:seed
```

---

## Стъпка 3: Стартирайте сървъра за разработка

```bash
pnpm run dev
```

Проверете в браузъра:

- [ ] Началната страница се зарежда на `http://localhost:3000`
- [ ] Можете да създадете акаунт
- [ ] Можете да влизате/излизате
- [ ] API документацията е достъпна на `http://localhost:3000/api/reference`

---

## Стъпка 4: Разберете структурата на проекта

```
directory-web-template/
├── apps/

│   │   ├── app/

```

---

## Стъпка 5: Работен процес за разработка

### 5.1 Създайте функционален клон

```bash
git checkout main
git pull origin main
git checkout -b feature/ime-na-funktsiya
```

### 5.2 Потвърдете и изпратете

```bash
git add .
git commit -m "feat: добавяне на система за известия на потребителите"
git push origin feature/ime-na-funktsiya
```
