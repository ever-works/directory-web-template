## 🔧 Fehlerbehebung

### Häufige Probleme

#### Installationsprobleme

- **Node.js-Version**: Stellen Sie sicher, dass Sie Node.js 18+ verwenden
- **Abhängigkeiten**: Führen Sie `npm install` oder `yarn install` im Website-Verzeichnis aus
- **Portkonflikte**: Verwenden Sie das Flag `--port` , um einen anderen Port anzugeben

#### Build-Probleme

- **TypeScript-Fehler**: Überprüfen Sie Ihre TypeScript-Konfiguration
- **Fehlende Abhängigkeiten**: Stellen Sie sicher, dass alle Pakete ordnungsgemäß installiert sind
- **Umgebungsvariablen**: Überprüfen Sie Ihre `.env` -Dateikonfiguration

#### Laufzeitprobleme

- **API-Verbindung**: Überprüfen Sie Ihre Umgebungsvariable `NEXT_PUBLIC_API_BASE_URL` - **Authentifizierung**: Überprüfen Sie die Anmeldeinformationen Ihres OAuth-Anbieters
- **Datenbank**: Stellen Sie sicher, dass Ihre Datenbankverbindung funktioniert

### Debug-Modus

Aktivieren Sie die Debug-Protokollierung, indem Sie Umgebungsvariablen festlegen:

```bash
DEBUG=directory-web-template:*
NODE_ENV=development
```
