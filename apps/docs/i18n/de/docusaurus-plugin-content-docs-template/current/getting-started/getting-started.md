# Erste Schritte mit dem Template

Willkommen beim Ever Works Directory Web Template. Dieser Abschnitt führt Sie durch jeden Schritt, von einem frischen Klon bis zu einer laufenden Anwendung.

## Was ist das Ever Works Template?

Eine voll ausgestattete Next.js-Verzeichniswebsite in einem Turborepo-Monorepo mit pnpm-Workspaces. Enthält Authentifizierung, Zahlungen, Admin-Dashboard, i18n, Git-basiertes CMS und mehr.

Wichtige Details:
- Paketname: directory-web-template
- Lizenz: AGPL-3.0
- Node.js-Anforderung: >= 20.19.0
- Paketmanager: pnpm mit Turborepo
- Framework: Next.js 16 mit React 19
- Datenbank-ORM: Drizzle ORM mit PostgreSQL

## Was Sie lernen werden

1. Installieren Sie das Template und alle Abhängigkeiten
2. Konfigurieren Sie Ihre Umgebung mit erforderlichen Secrets und Datenbank
3. Führen Sie die Anwendung lokal aus
4. Stellen Sie die Anwendung in der Produktion auf Vercel, Docker oder selbst gehostet bereit
5. Schlagen Sie Befehle mit der Kurzreferenz-Übersicht nach

## Abschnitts-Übersicht

### 1. Installation
Richten Sie Node.js (>= 20.19.0), pnpm ein und klonen Sie das Monorepo.

### 2. Umgebungskonfiguration
Konfigurieren Sie apps/web/.env.local mit allen erforderlichen und optionalen Variablen.

### 3. Schnellstart
Starten Sie Ihren ersten lokalen Entwicklungsserver in wenigen Minuten.

### 4. Erste Bereitstellung
Stellen Sie die Anwendung in der Produktion auf Vercel oder Docker bereit.

### 5. Kurzreferenz
Befehlsübersicht für häufige Aufgaben.
