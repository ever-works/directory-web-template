## 🔧 Solución de problemas

### Problemas comunes

#### Problemas de instalación

- **Versión de Node.js**: asegúrese de estar usando Node.js 18+
- **Dependencias**: Ejecute `npm install` o `yarn install` en el directorio del sitio web
- **Conflictos de puerto**: use el indicador `--port` para especificar un puerto diferente

#### Problemas de compilación

- **Errores de TypeScript**: verifique su configuración de TypeScript
- **Dependencias faltantes**: asegúrese de que todos los paquetes estén instalados correctamente
- **Variables de entorno**: Verifique la configuración de su archivo `.env` #### Problemas de tiempo de ejecución

- **Conexión API**: Verifique su variable de entorno `NEXT_PUBLIC_API_BASE_URL` - **Autenticación**: verifique las credenciales de su proveedor de OAuth
- **Base de datos**: asegúrese de que la conexión de su base de datos esté funcionando

### Modo de depuración

Habilite el registro de depuración configurando variables de entorno:

```bash
DEBUG=directory-web-template:*
NODE_ENV=development
```
