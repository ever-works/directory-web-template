# Guía de Capacitación del Equipo

Un programa de capacitación integral para incorporar y actualizar equipos de desarrollo en la plataforma Ever Works.

## 🎯 Objetivos de la Capacitación

Al finalizar este programa, los miembros del equipo serán capaces de:

- ✅ Configurar su entorno de desarrollo
- ✅ Comprender la arquitectura de Ever Works y el stack tecnológico
- ✅ Seguir estándares y convenciones de codificación establecidos
- ✅ Documentar APIs con el sistema de documentación automatizado
- ✅ Implementar funcionalidades siguiendo mejores prácticas
- ✅ Depurar y resolver problemas comunes
- ✅ Contribuir a la base de código con confianza

## 📚 Módulos de Capacitación

### 1. [Incorporación](/team-training/onboarding)

**Duración**: 1–2 días  
**Público**: Nuevos miembros del equipo

### 2. [Documentación API](/team-training/api-documentation)

**Duración**: 2–3 días  
**Público**: Todos los desarrolladores que trabajan en endpoints API

### 3. [Mejores Prácticas](/team-training/best-practices)

**Duración**: 1–2 días  
**Público**: Todos los miembros del equipo

### 4. [Ejercicios Prácticos](/team-training/exercises)

**Duración**: 3–5 días  
**Público**: Todos los miembros del equipo

## 📅 Calendario de Capacitación

### Nuevos Miembros del Equipo (2 semanas)

**Semana 1: Fundamentos**
- Días 1–2: [Incorporación](/team-training/onboarding)
- Días 3–4: [Documentación API](/team-training/api-documentation)
- Día 5: [Mejores Prácticas](/team-training/best-practices)

**Semana 2: Práctica**
- Días 1–3: [Ejercicios Prácticos](/team-training/exercises)
- Días 4–5: Primera tarea real con mentoría

## 📋 Prerrequisitos

### Conocimiento Requerido

- **JavaScript/TypeScript**: Intermedio a avanzado
- **React**: Sólida comprensión de hooks, componentes, gestión de estado
- **Next.js**: Familiaridad básica (App Router preferido)
- **Git**: Conceptos básicos de control de versiones

### Herramientas a Instalar

- Node.js 20.19.0+ (usa nvm)
- pnpm ([instalación](https://pnpm.io/installation))
- PostgreSQL 14+
- Git
- VS Code (o IDE preferido)

## 🆘 Obtener Ayuda

**Durante la Capacitación**:
- Pregunta al mentor o líder técnico
- Consulta la [Guía de Referencia Rápida](/getting-started/quick-reference)

**Después de la Capacitación**:
- Canales de Slack del equipo
- Retroalimentación de revisión de código
- Reuniones semanales del equipo
