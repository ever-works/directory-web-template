# Documentation Ever Works

Bienvenue dans la documentation d'**Ever Works** — un écosystème moderne de sites web d'annuaires composé de la **Plateforme Ever Works**, du **Modèle de site Ever Works** et du service SaaS **Pinler.com**.

## Qu'est-ce qu'Ever Works ?

Ever Works est une solution complète pour créer et gérer des sites web d'annuaires professionnels. Elle comprend trois composants principaux :

- **[Plateforme Ever Works](/platform)** — La plateforme backend qui alimente les sites d'annuaires, avec une API NestJS, un tableau de bord web Next.js, des outils CLI, des agents IA et un système de plugins.
- **[Modèle de site Ever Works](/template)** — Un modèle de site web d'annuaire full-stack prêt pour la production, construit avec Next.js 15, React 19, TypeScript et Tailwind CSS.
- **[Pinler.com](https://pinler.com)** — Un service SaaS construit sur la plateforme et le modèle Ever Works.
