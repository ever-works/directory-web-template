# Modèle de site Ever Works

Le **Modèle de site Ever Works** est une solution de site web d'annuaire full-stack, prête pour la production, construite avec des technologies web modernes. Il fournit tout ce dont vous avez besoin pour lancer rapidement un site d'annuaire professionnel.

## Fonctionnalités clés

- **Stack technologique moderne** — Next.js 15, React 19, TypeScript, Tailwind CSS, HeroUI React
- **Authentification flexible** — NextAuth.js v5, Supabase Auth, fournisseurs OAuth
- **Intégration de paiement** — Stripe, LemonSqueezy, Polar, gestion des abonnements
- **Internationalisation** — Plusieurs langues avec support RTL
- **CMS basé sur Git** — Synchronisation du contenu depuis des dépôts Git
- **Système de thèmes** — Thèmes intégrés avec génération dynamique de couleurs
- **Analytique et monitoring** — PostHog, Sentry, suivi des performances

## Cas d'utilisation

Le modèle est idéal pour les annuaires d'outils, les places de marché de services, les catalogues de ressources, les annuaires professionnels, les vitrines de produits et les plateformes communautaires.
