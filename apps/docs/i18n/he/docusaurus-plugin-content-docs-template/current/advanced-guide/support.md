## 🔧 פתרון בעיות

### בעיות נפוצות

#### בעיות התקנה

- **גרסת Node.js**: ודא שאתה משתמש ב-Node.js 18+
- **תלות**: הפעל את `npm install` או `yarn install` בספריית האתר
- **התנגשויות יציאה**: השתמש בדגל `--port` כדי לציין יציאה אחרת

#### בעיות בבנייה

- **שגיאות TypeScript**: בדוק את תצורת ה-TypeScript שלך
- **התלות חסרות**: ודא שכל החבילות מותקנות כהלכה
- **משתני סביבה**: אמת את תצורת הקובץ `.env` שלך

#### בעיות בזמן ריצה

- **חיבור API**: בדוק את משתנה הסביבה `NEXT_PUBLIC_API_BASE_URL` שלך
- **אימות**: אמת את האישורים של ספק ה-OAuth שלך
- **מסד נתונים**: ודא שחיבור מסד הנתונים שלך פועל

### מצב ניפוי באגים

אפשר רישום באגים על ידי הגדרת משתני סביבה:

```bash
DEBUG=directory-web-template:*
NODE_ENV=development
```
