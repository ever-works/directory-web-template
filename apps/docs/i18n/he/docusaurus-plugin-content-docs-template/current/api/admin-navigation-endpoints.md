# נקודות קצה של ניווט מנהל ואינדקס מיקום

נקודות קצה אלו של מנהל מערכת מנהלות קישורי ניווט מותאמים אישית באתר ואת אינדקס המיקום הגיאוגרפי. נקודות קצה ניווט מאפשרות להגדיר קישורי כותרת עליונה ותחתית מותאמות אישית המאוחסנים ב-`config.yml`. נקודות קצה של אינדקס מיקום מנהלות את האינדקס המרחבי המשמש לניתוח גיאוגרפי ותכונות מפה.

## סקירה כללית

|נקודת קצה|שיטה|Auth|תיאור|
|---|---|---|---|
|`/api/admin/navigation`|קבל|מנהל מערכת|קבל תצורת ניווט מותאמת אישית|
|`/api/admin/navigation`|תיקון|מנהל מערכת|עדכן פריטי ניווט מותאמים אישית|
|`/api/admin/location-index`|קבל|מנהל מערכת|קבל נתונים סטטיסטיים של אינדקס מיקום|
|`/api/admin/location-index`|פוסט|מנהל מערכת|בנה מחדש או נקה את אינדקס המיקום|

## נקודות קצה של ניווט

### קבל תצורת ניווט

```
GET /api/admin/navigation
```

מאחזר את פריטי הניווט `custom_header` ו-`custom_footer` מהקובץ `config.yml` של האתר. מחזירה מערכים ריקים אם לא הוגדר ניווט מותאם אישית.

**אימות:** נדרש מנהל מערכת (דרך `getCachedApiSession`)

**תגובת הצלחה (200):**

```json
{
  "custom_header": [
    {
      "label": "About",
      "path": "/about"
    },
    {
      "label": "Documentation",
      "path": "/pages/docs"
    }
  ],
  "custom_footer": [
    {
      "label": "GitHub",
      "path": "https://github.com/example"
    },
    {
      "label": "footer.PRIVACY_POLICY",
      "path": "/pages/privacy-policy"
    }
  ]
}
```

לכל פריט ניווט שני שדות:

|שדה|הקלד|תיאור|
|---|---|---|
|`label`|מחרוזת|טקסט תצוגה (טקסט רגיל או מפתח תרגום i18n כמו `"footer.PRIVACY_POLICY"`)|
|`path`|מחרוזת|נתיב כתובת אתר (מסלול פנימי שמתחיל ב-`/` או כתובת אתר חיצונית ב-`http://`/`https://`)|

|סטטוס|מצב|
|---|---|
| 401 |לא מאומת כמנהל|
| 500 |קריאת התצורה נכשלה|

**מקור:** `template/app/api/admin/navigation/route.ts`

### עדכן את תצורת הניווט

```
PATCH /api/admin/navigation
```

מעדכן את פריטי הניווט המותאמים אישית בכותרת העליונה או התחתונה ב-`config.yml`. מאמת את פורמט הנתיב של כל פריט כדי למנוע התקפות XSS באמצעות ערכות URL מסוכנות.

**אימות:** נדרש מנהל מערכת

**גוף הבקשה:**

```json
{
  "type": "header",
  "items": [
    {
      "label": "About",
      "path": "/about"
    },
    {
      "label": "Blog",
      "path": "https://blog.example.com"
    }
  ]
}
```

|שדה|הקלד|חובה|תיאור|
|---|---|---|---|
|`type`|מחרוזת|כן|`"header"` או `"footer"`|
|`items`|מערך|כן|מערך פריטי ניווט|
|`items[].label`|מחרוזת|כן|תווית תצוגה לא ריקה|
|`items[].path`|מחרוזת|כן|נתיב כתובת אתר חוקי|

**אימות נתיב:**

הפונקציה `isValidNavigationPath()` אוכפת כללי פורמט נתיב קפדניים:

|פורמט נתיב|מותר|דוגמה|
|---|---|---|
|מסלולים פנימיים|כן|`/about`, `/pages/docs`|
|כתובות אתרים של HTTPS|כן|`https://example.com`|
|כתובות URL של HTTP|כן|`http://example.com`|
|כתובות אתרים הקשורות לפרוטוקול|לא|`//evil.com`|
|כתובות URL של JavaScript|לא|`javascript:alert(1)`|
|כתובות אתרים של נתונים|לא|`data:text/html,...`|
|תוכניות אחרות|לא|`vbscript:`, `file:`|

**תגובת הצלחה (200):**

```json
{
  "success": true,
  "type": "header",
  "items": [
    {
      "label": "About",
      "path": "/about"
    }
  ]
}
```

**תגובות שגיאה:**

|סטטוס|מצב|
|---|---|
| 400 |`type` אינו `"header"` או `"footer"`|
| 400 |`items` אינו מערך|
| 400 |חסרים בפריט שדות מחרוזת `label` או `path`|
| 400 |פורמט נתיב לא חוקי (מניעת XSS)|
| 401 |לא מאומת כמנהל|
| 500 |כתיבת התצורה נכשלה|

העבר מערך `items` ריק כדי לנקות את כל הניווט המותאם אישית עבור הסוג שצוין.

**מקור:** `template/app/api/admin/navigation/route.ts`

## נקודות קצה של אינדקס מיקום

### קבל סטטיסטיקות של אינדקס מיקום

```
GET /api/admin/location-index
```

מחזיר נתונים סטטיסטיים על אינדקס המיקום הגיאוגרפי, כולל סך כל הפריטים שנוספו לאינדקס, ספירת ערים ומדינה, ובניית מטא נתונים מחדש. משתמש בשירות אינדקס המיקום לאחזור נתונים.

**אימות:** נדרש מנהל מערכת (דרך `checkAdminAuth()`)

**מטמון:** מושבת -- משתמש ב-`force-dynamic`, `revalidate: 0` ו-`force-no-store`.

**תגובת הצלחה (200):**

```json
{
  "success": true,
  "data": {
    "totalIndexed": 450,
    "citiesCount": 85,
    "countriesCount": 25,
    "remoteCount": 30,
    "lastIndexedAt": "2024-01-20T10:30:00.000Z",
    "lastRebuildAt": "2024-01-15T08:00:00.000Z"
  }
}
```

|סטטוס|מצב|
|---|---|
| 401 |לא מאומת כמנהל|
| 500 |שגיאת שרת פנימית|

**מקור:** `template/app/api/admin/location-index/route.ts`

### ניהול אינדקס מיקום

```
POST /api/admin/location-index
```

מבצע פעולות ניהול באינדקס המיקום. תומך בבנייה מחדש של האינדקס מאפס או בניקוי כל הערכים.

**אימות:** נדרש מנהל מערכת

**גוף הבקשה:**

```json
{
  "action": "rebuild"
}
```

|שדה|הקלד|חובה|תיאור|
|---|---|---|---|
|`action`|מחרוזת|כן|`"rebuild"` או `"clear"`|

**פעולות:**

|פעולה|תיאור|
|---|---|
|`rebuild`|שולף את כל הפריטים מהמאגר ומוסיף מחדש את נתוני המיקום שלהם. מחזיר סטטיסטיקות בנייה מחדש.|
|`clear`|מסיר את כל הערכים מאינדקס המיקום. מחזירה את מספר הערכים שנמחקו.|

**תגובת הצלחה (200) -- בנייה מחדש:**

```json
{
  "success": true,
  "data": {
    "indexed": 420,
    "skipped": 80,
    "errors": 0
  }
}
```

**תגובת הצלחה (200) -- ברור:**

```json
{
  "success": true,
  "data": {
    "cleared": 450
  }
}
```

**תגובות שגיאה:**

|סטטוס|מצב|
|---|---|
| 400 |פעולה לא חוקית (לא `"rebuild"` או `"clear"`)|
| 401 |לא מאומת כמנהל|
| 500 |שגיאת שרת פנימית|

**מקור:** `template/app/api/admin/location-index/route.ts`

## פרטי יישום מרכזיים

- **מניעת XSS:** אימות נתיב ניווט דוחה את כל סכימות כתובת האתר למעט `/`, `http://` ו-`https://`. פעולה זו חוסמת את `javascript:`, `data:`, `vbscript:`, וכתובות URL-יחסיות לפרוטוקול (`//evil.com`) שניתן להשתמש בהן עבור סקריפטים חוצי אתרים.
- **אחסון תצורה:** פריטי ניווט מאוחסנים ב-`config.yml` תחת המקשים `custom_header` ו-`custom_footer`, מתמידים באמצעות `configManager.updateNestedKey()`.
- **i18n תוויות:** תוויות ניווט יכולות להיות טקסט רגיל או מפתחות תרגום (לדוגמה, `"footer.PRIVACY_POLICY"`). ה-frontend אחראי על פתרון מפתחות תרגום.
- **בנייה מחדש של אינדקס מיקום:** פעולת הבנייה מחדש טוענת את כל הפריטים מה-`ItemRepository` ומעבירה אותם לשירות אינדקס המיקום. זו יכולה להיות פעולה עתירת משאבים עבור מערכי נתונים גדולים.
- **פריצת מטמון:** נקודות קצה של אינדקס מיקום משביתות במפורש את כל האחסון במטמון כדי להבטיח שמרכז השליטה של המנהל יציג תמיד נתונים עדכניים.
