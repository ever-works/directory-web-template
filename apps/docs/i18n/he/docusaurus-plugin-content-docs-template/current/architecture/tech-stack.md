# מחסנית טכנולוגיה

מסמך זה מספק סקירה מקיפה של כל הטכנולוגיות המשמשות ב-Ever Works.

## דרישות מערכת

- **Node.js**: 20.19.0 ומעלה
- **PostgreSQL**: 14.0 ומעלה
- **מנהל החבילות**: npm, pnpm, yarn, or bun

## Backend Technologies

### מסד נתונים ו-ORM

- **[PostgreSQL 14+](https://www.postgresql.org/)** - מסד נתונים יחסי
  - תאימות ל-ACID
  - תכונות מתקדמות (JSONB, חיפוש בטקסט מלא)
  - ביצוע מעולה
  - תמיכה ב-JSON
  - טריגרים ונהלים מאוחסנים

- **[Drizzle ORM 0.40.0](https://orm.drizzle.team/)** - TypeScript ORM
  - שאילתות בטוחות לפי סוג
  - תקורה מינימלית
  - תחביר דמוי SQL
  - מערכת הגירה
  - שאילתות קשר
  - הצהרות מוכנות

- **[Supabase](https://supabase.com/)** - Backend-as-a-Service (אופציונלי)
  - מארח את PostgreSQL
  - מנויים בזמן אמת
  - אבטחה ברמת השורה
  - אישור מובנה
  - דליי אחסון
  - פונקציות קצה

### אימות

- **[NextAuth.js 5.0 (ביטא)](https://authjs.dev/)** - ספריית אימות
  - מספר ספקי OAuth (Google, GitHub, Facebook, Twitter)
  - מפגשי JWT ומסד נתונים
  - תמיכה ב-TypeScript
  - שיטות עבודה מומלצות לאבטחה
  - אימות מבוסס אישורים
  - ניהול מפגשים

- **[Supabase Auth](https://supabase.com/auth)** - פתרון אימות חלופי
  - ניהול משתמשים מובנה
  - ספקים חברתיים
  - אימות דוא"ל
  - איפוס סיסמה
  - קישורי קסם
  - אישור טלפון

### Dual Auth Architecture

Ever Works תומך ב-**ב-NextAuth.js וב-Supabase Auth** בו-זמנית:

- NextAuth עבור זרימות OAuth מסורתיות
- Supabase Auth לתכונות בזמן אמת
- ניהול מושב מאוחד
- החלפת ספק חלקה

- **Next.js 16**: כאשר זמין
- **TypeScript 5.x**: התכונות האחרונות
- **Node.js 22**: שדרוג LTS

### תוספות פוטנציאליות

- **GraphQL**: לדרישות נתונים מורכבות
- **WebSockets**: תכונות בזמן אמת
- **PWA**: תכונות אפליקציית אינטרנט מתקדמות
- **מחשוב קצה**: ביצועים משופרים

## מטריצת החלטות טכנולוגיות

|דרישה|בחירת טכנולוגיה|נימוק|
|-------------|-------------------|-----------|
|**מסגרת**|Next.js 15|מסגרת React הטובה מסוגה עם נתב אפליקציות|
|**מסד נתונים**|PostgreSQL + טפטוף|בטוח לסוג, ביצועים, ניתן להרחבה|
|**אישור**|NextAuth.js + Supabase|גמישות ספק כפול|
|**סטיילינג**|Tailwind CSS + HeroUI|פיתוח מהיר, עיצוב עקבי|
|**מדינה**|Zustand + שאילתת תגובה|מצב לקוח פשוט + מצב שרת חזק|
|**טפסים**|טופס הוק תגובה + זוד|ביצועים + בטיחות סוג|
|**i18n**|next-intl|התמיכה הטובה ביותר בנתב אפליקציית Next.js|
|**תשלום**|Stripe + LemonSqueezy|גמישות + תאימות גלובלית|
|**מייל**|שלח שוב + Novu|ידידותי למפתחים + רב ערוצי|
|**אנליטיקה**|PostHog + Sentry|תובנות מוצר + מעקב שגיאות|
