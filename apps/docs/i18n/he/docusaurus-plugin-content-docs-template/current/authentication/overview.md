# סקירת אימות

Ever Works מספקת מערכת אימות גמישה ומאובטחת התומכת במספר ספקים ושיטות אימות.

## ארכיטקטורת האימות

התבנית משתמשת בגישת אימות היברידית, התומכת בו-זמנית ב-NextAuth.js ו-Supabase Auth, המאפשרת לך לבחור את הפתרון המתאים ביותר לצרכיך.

## שיטות אימות נתמכות

### 1. ספקי OAuth

NextAuth.js OAuth תומך ב: Google, GitHub, Facebook, Twitter/X, Microsoft  
Supabase OAuth תומך ב: Google, GitHub, Facebook, Twitter/X, Discord, Apple

### 2. אימות בדוא"ל/סיסמה

NextAuth.js Credentials: דוא"ל וסיסמה מותאמים אישית, גיבוב bcrypt, אחסון הפעלות בבסיס הנתונים  
Supabase Auth: דוא"ל/סיסמה מובנה, אימות דוא"ל, איפוס סיסמה

### 3. קישורים קסומים
Supabase Auth: אימות ללא סיסמה באמצעות קישורים קסומים בדוא"ל

### 4. WebAuthn / מפתחות גישה
NextAuth.js: אימות ביומטרי, מפתחות אבטחת חומרה, FIDO2

## ניהול הפעלות
אסימוני JWT לאימות חסר מצב, הפעלות בסיס נתונים למצב מתמיד, טיפול מאובטח בעוגיות, רענון אסימון אוטומטי

## תכונות אבטחה
הגנת CSRF, הגבלת קצב, הגנה מפני כוח גס, גיבוב סיסמאות מאובטח עם bcrypt
