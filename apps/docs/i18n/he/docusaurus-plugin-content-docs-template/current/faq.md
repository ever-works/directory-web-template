# שאלות נפוצות

## מה זה Directory Web Template?

Directory Web Template הוא פתרון מוכן לייצור לאתר ספרייה בנוי על Next.js, המיועד להפעלת ספריות כלים, שירותים או מוצרים.

## האם ניתן להשתמש ב-Template ללא Ever Works Platform?

כן, התבנית פועלת באופן עצמאי לחלוטין ממנגנון Ever Works.

## מה המחסנית הטכנולוגית?

- **Next.js 15** – מסגרת React
- **React 19** – ספריית UI
- **TypeScript 5** – תכנות מוקלד
- **Tailwind CSS** – מסגרת CSS
- **Drizzle ORM** – ניהול מסד נתונים
- **NextAuth.js** – אימות

## אילו ספקי OAuth נתמכים?

- Google
- GitHub
- Facebook
- Twitter
- Microsoft

## אילו ספקי תשלומים נתמכים?

- Stripe
- LemonSqueezy
- Polar

## מהי פלטפורמת הפריסה המומלצת?

אנו ממליצים על **Vercel** לפריסה הקלה והמהירה ביותר.

## איזה מסד נתונים מומלץ?

**PostgreSQL** דרך Supabase לסביבת ייצור.

## כיצד עובדת מערכת ניהול התוכן?

התוכן מנוהל דרך קבצי YAML/Markdown במאגר Git שנשכפל לתיקיית `.content/`.
