# תחילת עבודה עם התבנית

ברוכים הבאים ל-**Directory Web Template**! מדריך זה ילווה אתכם מ-clone של המאגר ועד להפעלת אפליקציה עובדת.

## מה זה Ever Works Template?

Ever Works Template הוא ספריית Next.js עם כל הפיצ'רים, מאורגנת ב-Turborepo Monorepo. מספק את כל מה שנדרש להפעלת ספריית כלים, שירותים או מוצרים.

## דרישות מוקדמות

- **Node.js >= 20.19.0**
- **pnpm** – מנהל חבילות
- **Next.js 16** – מסגרת React
- **Drizzle ORM** – ניהול מסד נתונים
