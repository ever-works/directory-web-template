# מדריך מהיר

## פקודות עיקריות

| פקודה | תיאור |
|-------|-------|
| `pnpm run dev` | הפעלת שרת פיתוח |
| `pnpm run build` | בנייה לסביבת ייצור |
| `pnpm run lint` | בדיקת שגיאות קוד |
| `pnpm run db:generate` | יצירת מיגרציות למסד נתונים |
| `pnpm run db:migrate` | הפעלת מיגרציות |
| `pnpm run db:seed` | טעינת נתונים ראשוניים |

## קבצים ותיקיות מרכזיים

| קובץ/תיקייה | תיאור |
|-------------|-------|
| `.env.local` | משתני סביבה מקומיים |
| `.content/` | תוכן ששוכפל מ-Git CMS |
| `app/` | מסלולי Next.js App Router |
| `lib/` | לוגיקת עסקים ושירותים |
| `components/` | רכיבי React |
