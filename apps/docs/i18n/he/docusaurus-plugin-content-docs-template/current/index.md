# Directory Web Template

**Directory Web Template** היא פתרון מודרני ומלא לאתר ספרייה, בנוי עם Next.js 16. מתוכנן לאתרי ספרייה מקצועיים לכלים, שירותים, מוצרים או פלטפורמות רישום.

## תכונות עיקריות

- **מחסנית טכנולוגית מודרנית** – Next.js 16, React 19, TypeScript
- **Turborepo Monorepo** – מבנה מאורגן של פרויקט
- **אימות גמיש** – NextAuth.js v5, ספקי OAuth
- **שילוב תשלומים** – Stripe, LemonSqueezy, Polar
- **בינאום** – תמיכה רב-לשונית
- **מערכת ניהול תוכן מבוססת Git** – ניהול תוכן דרך Git
- **מערכת ערכות נושא** – עיצוב מותאם אישית
- **ניתוחים ומעקב** – אנליטיקס מובנה
- **לוח בקרה ניהולי** – ניהול תוכן
- **אופטימיזציה למנועי חיפוש** – מותאם למנועי חיפוש

## התחלה מהירה

```bash
git clone https://github.com/ever-works/directory-web-template.git
cd directory-web-template
pnpm install
pnpm run dev
```
