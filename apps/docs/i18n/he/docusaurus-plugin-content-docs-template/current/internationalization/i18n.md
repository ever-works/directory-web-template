# בינאום (i18n)

Ever Works Template משתמש ב-`next-intl` לבינאום, ותומך ב-21 לוקאלים עם ניתוב מבוסס-URL, זיהוי אוטומטי מהדפדפן, הודעות גיבוי, פריסות מימין לשמאל ותגיות hreflang לקידום אתרים. דף זה מתעד את ארכיטקטורת ה-i18n, התצורה ודפוסי השימוש.

## ארכיטקטורה

| קובץ | מטרה |
|------|---------|
| `i18n/routing.ts` | תצורת ניתוב (לוקאלים נתמכים, אסטרטגיית קידומת) |
| `i18n/request.ts` | זיהוי לוקאל וטעינת הודעות לכל בקשה |
| `i18n/navigation.ts` | Link, redirect, usePathname, useRouter עם תמיכת לוקאל |
| `lib/constants.ts` | רשימת שפות, ברירת מחדל, הגדרות RTL |
| `lib/seo/hreflang.ts` | יצירת תגיות hreflang לקידום אתרים |
| `components/language-switcher.tsx` | מחליף שפה בממשק |
| `messages/*.json` | קבצי תרגום (אחד לכל שפה) |
| `app/[locale]/layout.tsx` | פריסת שורש עם NextIntlClientProvider |

## שפות נתמכות

התבנית מגדירה 21 לוקאלים ב-`lib/constants.ts`:

```typescript
export const DEFAULT_LOCALE = 'en';
export const LOCALES = [
  'en', 'fr', 'es', 'de', 'zh', 'ar', 'he',
  'ru', 'uk', 'pt', 'it', 'ja', 'ko', 'nl',
  'pl', 'tr', 'vi', 'th', 'hi', 'id', 'bg'
] as const;

export type Locale = (typeof LOCALES)[number];

/** לוקאלים המשתמשים בכיוון טקסט מימין לשמאל */
export const RTL_LOCALES: readonly Locale[] = ['ar', 'he'] as const;
```

| `fr` | צרפתית | LTR |
| `es` | ספרדית | LTR |

## תצורת ניתוב

מודול הניתוב ב-`i18n/routing.ts` מגדיר את המיפוי משפות ל-URL:

```typescript
import { defineRouting } from "next-intl/routing";
import { DEFAULT_LOCALE, LOCALES } from "@/lib/constants";

export const routing = defineRouting({
  locales: LOCALES,
  defaultLocale: DEFAULT_LOCALE,
  localeDetection: true,
  localePrefix: "as-needed",
});
```

### אסטרטגיית קידומת

אסטרטגיית `"as-needed"` שומרת על כתובות URL נקיות עבור ברירת המחדל:

| URL | לוקאל שזוהה | הסבר |
|-----|-----------------|-------------|
| `/about` | `en` (ברירת מחדל) | אין צורך בקידומת לשפת ברירת המחדל |
| `/fr/about` | `fr` | קידומת נדרשת לשפות אחרות |
| `/ar/about` | `ar` | קידומת נדרשת לשפות אחרות |

### זיהוי שפה

עם `localeDetection: true`, המערכת בודקת אוטומטית:

1. עוגיית שפה (מוגדרת על ידי רכיב LanguageSwitcher)
2. כותרת HTTP `Accept-Language` מהדפדפן
3. חזרה ל-`DEFAULT_LOCALE` (`'en'`) אם לא נמצאה התאמה

## טעינת הודעות וגיבוי

מודול `i18n/request.ts` מתבצע בכל בקשת שרת:

```typescript
export default getRequestConfig(async ({ requestLocale }) => {
  let locale = await requestLocale;

  if (!locale || !routing.locales.includes(locale as any)) {
    locale = routing.defaultLocale;
  }

  const userMessages = (await import(`../messages/${locale}.json`)).default;
  const defaultMessages = (await import(`../messages/en.json`)).default;
  const messages = deepmerge(defaultMessages, userMessages);

  return { locale, messages };
});
```

אסטרטגיית `deepmerge` מבטיחה שמפתחות לא מתורגמים מציגים טקסט אנגלי.

## קבצי תרגום

הודעות תרגום מאוחסנות כקבצי JSON בתיקיית `messages/`:

```
messages/
  en.json    # אנגלית (מקור, כל המפתחות נוכחים)
  fr.json    # צרפתית
  he.json    # עברית
  ... (21 קבצים בסך הכל)
```

### שימוש בתרגומים ברכיבים

**רכיבי לקוח** משתמשים ב-hook `useTranslations`:

```tsx
import { useTranslations } from "next-intl";

function LoginButton() {
  const t = useTranslations("auth");
  return <button>{t("SIGN_IN")}</button>;
}
```

**רכיבי שרת** משתמשים ב-`getTranslations` האסינכרוני:

```tsx
import { getTranslations } from "next-intl/server";

export default async function Page() {
  const t = await getTranslations("common");
  return <h1>{t("DISCOVER")}</h1>;
}
```

## הוספת שפה חדשה

1. הוסף את קוד השפה למערך `LOCALES` ב-`lib/constants.ts`
2. אם השפה היא RTL — הוסף אותה ל-`RTL_LOCALES`
3. צור `messages/<קוד>.json` עם מפתחות מתורגמים
4. הוסף SVG דגל ב-`public/flags/<קוד>.svg`
5. הוסף ערך שפה ל-`languageMap` ב-`components/language-switcher.tsx`
6. הוסף מיפוי hreflang ב-`lib/seo/hreflang.ts`

## שיטות עבודה מומלצות

1. **השתמש ב-`Link` מ-`@/i18n/navigation`** במקום `next/link`
2. **הוסף מפתחות חדשים תחילה ל-`en.json`** שכן זו שפת המקור
3. **לעולם אל תקודד מחרוזות אנגליות** ברכיבים
4. **הסתמך על גיבוי `deepmerge`** — קבצים מתורגמים חלקית פועלים כראוי
5. **בדוק פריסות RTL** על ידי מעבר לערבית או עברית
