# מדריך הדרכת צוות

תוכנית הכשרה מקיפה לאינדוקציה ופיתוח מיומנויות של צוותי פיתוח בפלטפורמת Ever Works.

## 🎯 מטרות ההכשרה

לאחר השלמת תוכנית ההכשרה, חברי הצוות יוכלו:

- ✅ להגדיר סביבת פיתוח עובדת
- ✅ להבין את ארכיטקטורת Ever Works ומחסנית הטכנולוגיות
- ✅ לעקוב אחר תקני קוד ומוסכמות מבוססות
- ✅ לתעד API באמצעות מערכת התיעוד האוטומטית
- ✅ לממש תכונות לפי שיטות עבודה מומלצות
- ✅ לאבחן ולפתור בעיות נפוצות
- ✅ לתרום לבסיס הקוד בביטחון

## 📚 מודולי הכשרה

### 1. [מדריך הכנסה לעבודה](/team-training/onboarding)

**משך**: 1–2 ימים  
**קהל יעד**: חברי צוות חדשים

### 2. [תיעוד API](/team-training/api-documentation)

**משך**: 2–3 ימים  
**קהל יעד**: כל המפתחים העובדים עם נקודות קצה API

### 3. [שיטות עבודה מומלצות](/team-training/best-practices)

**משך**: 1–2 ימים  
**קהל יעד**: כל חברי הצוות

### 4. [תרגולים מעשיים](/team-training/exercises)

**משך**: 3–5 ימים  
**קהל יעד**: כל חברי הצוות

## 📅 לוח זמנים הכשרה

### חברי צוות חדשים (שבועיים)

**שבוע 1: יסודות**
- ימים 1–2: [מדריך הכנסה לעבודה](/team-training/onboarding)
- ימים 3–4: [תיעוד API](/team-training/api-documentation)
- יום 5: [שיטות עבודה מומלצות](/team-training/best-practices)

**שבוע 2: תרגול**
- ימים 1–3: [תרגולים מעשיים](/team-training/exercises)
- ימים 4–5: משימה ראשונה אמיתית עם מנטור

## 📋 דרישות מוקדמות

### ידע נדרש

- **JavaScript/TypeScript**: בינוני ומעלה
- **React**: הבנה טובה של Hooks, רכיבים, ניהול מצב
- **Next.js**: היכרות בסיסית (App Router מועדף)
- **Git**: יסודות בקרת גרסאות

### כלים להתקנה

- Node.js 20.19.0+ (השתמש ב-nvm)
- pnpm ([התקנה](https://pnpm.io/installation))
- PostgreSQL 14+
- Git
- VS Code (או IDE מועדף)

## 🆘 קבלת עזרה

**במהלך ההכשרה**:
- שאל מנטור או מוביל טכני
- עיין ב[מדריך עזר מהיר](/getting-started/quick-reference)

**לאחר ההכשרה**:
- ערוצי Slack של הצוות
- משוב בסקירות קוד
- פגישות צוות שבועיות
