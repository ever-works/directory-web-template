## 🔧 Risoluzione dei problemi

### Problemi comuni

#### Problemi di installazione

- **Versione Node.js**: assicurati di utilizzare Node.js 18+
- **Dipendenze**: esegui `npm install` o `yarn install` nella directory del sito web
- **Conflitti porte**: usa il flag `--port` per specificare una porta diversa

#### Problemi di creazione

- **Errori TypeScript**: controlla la configurazione di TypeScript
- **Dipendenze mancanti**: assicurarsi che tutti i pacchetti siano installati correttamente
- **Variabili d'ambiente**: verifica la configurazione del file `.env` #### Problemi di runtime

- **Connessione API**: controlla la variabile di ambiente `NEXT_PUBLIC_API_BASE_URL` - **Autenticazione**: verifica le credenziali del tuo provider OAuth
- **Database**: assicurati che la connessione al database funzioni

### Modalità di debug

Abilita la registrazione del debug impostando le variabili di ambiente:

```bash
DEBUG=directory-web-template:*
NODE_ENV=development
```
