# Iniziare con il Template

Benvenuto nel Ever Works Directory Web Template. Questa sezione ti guida attraverso ogni passaggio, da un clone fresco a un'applicazione funzionante.

## Cos'è il Template Ever Works?

Un sito web directory Next.js completo in un monorepo Turborepo con workspace pnpm. Include autenticazione, pagamenti, dashboard amministratore, i18n, CMS basato su Git e molto altro.

Dettagli chiave:
- Nome pacchetto: directory-web-template
- Licenza: AGPL-3.0
- Requisito Node.js: >= 20.19.0
- Gestore pacchetti: pnpm con Turborepo
- Framework: Next.js 16 con React 19
- ORM Database: Drizzle ORM con PostgreSQL

## Cosa Imparerai

1. Installare il template e tutte le dipendenze
2. Configurare il tuo ambiente con i segreti richiesti e il database
3. Eseguire l'applicazione localmente
4. Distribuire in produzione su Vercel, Docker o self-hosted
5. Consultare i comandi con il foglio di riferimento rapido

## Panoramica della Sezione

### 1. Installazione
Configura Node.js (>= 20.19.0), pnpm e clona il monorepo.

### 2. Configurazione Ambiente
Configura apps/web/.env.local con tutte le variabili richieste e opzionali.

### 3. Avvio Rapido
Avvia il tuo primo server di sviluppo locale in pochi minuti.

### 4. Primo Deployment
Distribuisci in produzione su Vercel o Docker.

### 5. Guida Rapida
Foglio di riferimento dei comandi per le attività comuni.
