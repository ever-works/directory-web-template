## 🔧 Problemen oplossen

### Veelvoorkomende problemen

#### Installatieproblemen

- **Node.js-versie**: zorg ervoor dat u Node.js 18+ gebruikt
- **Afhankelijkheden**: voer `npm install` of `yarn install` uit in de websitemap
- **Poortconflicten**: gebruik de vlag `--port` om een andere poort op te geven

#### Bouwproblemen

- **TypeScript-fouten**: Controleer uw TypeScript-configuratie
- **Ontbrekende afhankelijkheden**: zorg ervoor dat alle pakketten correct zijn geïnstalleerd
- **Omgevingsvariabelen**: controleer uw `.env` -bestandsconfiguratie

#### Runtime-problemen

- **API-verbinding**: controleer uw omgevingsvariabele `NEXT_PUBLIC_API_BASE_URL` - **Authenticatie**: controleer de inloggegevens van uw OAuth-provider
- **Database**: Zorg ervoor dat uw databaseverbinding werkt

### Foutopsporingsmodus

Schakel logboekregistratie voor foutopsporing in door omgevingsvariabelen in te stellen:

```bash
DEBUG=directory-web-template:*
NODE_ENV=development
```
