# Aan de slag met het Template

Welkom bij het Ever Works Directory Web Template. Dit gedeelte leidt u door elke stap, van een nieuwe kloon tot een draaiende applicatie.

## Wat is het Ever Works Template?

Een volledig uitgeruste Next.js-directorywebsite in een Turborepo-monorepo met pnpm-workspaces. Bevat authenticatie, betalingen, beheerdashboard, i18n, Git-gebaseerd CMS en meer.

Belangrijke details:
- Pakketnaam: directory-web-template
- Licentie: AGPL-3.0
- Node.js-vereiste: >= 20.19.0
- Pakketbeheerder: pnpm met Turborepo
- Framework: Next.js 16 met React 19
- Database-ORM: Drizzle ORM met PostgreSQL

## Wat u leert

1. Installeer het template en alle afhankelijkheden
2. Configureer uw omgeving met vereiste geheimen en database
3. Voer de applicatie lokaal uit
4. Implementeer naar productie op Vercel, Docker of zelf gehost
5. Zoek opdrachten op met het snelreferentie-spiekbriefje

## Sectie-overzicht

### 1. Installatie
Stel Node.js (>= 20.19.0), pnpm in en kloon de monorepo.

### 2. Omgevingsconfiguratie
Configureer apps/web/.env.local met alle vereiste en optionele variabelen.

### 3. Snelstart
Start uw eerste lokale ontwikkelingsserver in enkele minuten.

### 4. Eerste implementatie
Implementeer naar productie op Vercel of Docker.

### 5. Snelreferentie
Opdrachtenoverzicht voor veelvoorkomende taken.
