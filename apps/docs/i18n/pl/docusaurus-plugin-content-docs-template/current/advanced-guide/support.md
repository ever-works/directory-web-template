## 🔧 Rozwiązywanie problemów

### Typowe problemy

#### Problemy z instalacją

- **Wersja Node.js**: Upewnij się, że używasz Node.js 18+
- **Zależności**: Uruchom `npm install` lub `yarn install` w katalogu strony internetowej
- **Konflikty portów**: Użyj flagi `--port` , aby określić inny port

#### Problemy z kompilacją

- **Błędy TypeScript**: Sprawdź konfigurację TypeScriptu
- **Brakujące zależności**: Upewnij się, że wszystkie pakiety są poprawnie zainstalowane
- **Zmienne środowiskowe**: Sprawdź konfigurację pliku `.env` #### Problemy z czasem wykonania

- **Połączenie API**: Sprawdź zmienną środowiskową `NEXT_PUBLIC_API_BASE_URL` - **Uwierzytelnianie**: Sprawdź poświadczenia dostawcy OAuth
- **Baza danych**: Upewnij się, że połączenie z bazą danych działa

### Tryb debugowania

Włącz rejestrowanie debugowania, ustawiając zmienne środowiskowe:

```bash
DEBUG=directory-web-template:*
NODE_ENV=development
```
