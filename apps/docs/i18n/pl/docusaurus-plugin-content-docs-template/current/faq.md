# Najczęściej zadawane pytania

## Ogólne

### Czym jest Directory Web Template?
Gotowe do produkcji, pełnostoikowe rozwiązanie Next.js dla stron katalogów. Sklonuj, dostosuj i wdróż, aby tworzyć profesjonalne strony katalogów.

### Czy mogę używać Template bez platformy Ever Works?
Tak. Template działa niezależnie. Platforma jest oddzielnym, opcjonalnym produktem.

## Stack technologiczny

### Jakich technologii używa Template?
- Framework: Next.js 15, React 19
- Język: TypeScript 5
- Stylowanie: Tailwind CSS 4, HeroUI React
- ORM: Drizzle ORM z PostgreSQL
- Auth: NextAuth.js v5
- Płatności: Stripe, LemonSqueezy, Polar

### Jakich dostawców uwierzytelniania obsługuje?
Google, GitHub, Facebook, Twitter i Microsoft via NextAuth.js v5.

### Jakich dostawców płatności obsługuje?
Stripe, LemonSqueezy i Polar.

## Wdrożenie

### Jak wdrożyć Template?
Zalecane: Vercel. Docker również obsługiwany.

### Jakiej bazy danych powinienem używać?
PostgreSQL, zazwyczaj via Supabase lub bezpośrednie połączenie.

## Treść

### Jak działa CMS oparty na Git?
Treść jest przechowywana w plikach YAML/Markdown w repozytorium Git, klonowanym podczas budowania do .content/.

### Czy mogę dodawać elementy ręcznie?
Tak, edytuj pliki YAML/Markdown w repozytorium danych CMS.
