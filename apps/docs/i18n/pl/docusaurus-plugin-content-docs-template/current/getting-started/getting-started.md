# Pierwsze kroki z Template

Witamy w Ever Works Directory Web Template. Od klonowania do uruchomionej aplikacji.

## Czym jest Ever Works Template?

Kompletna strona katalogów Next.js w monorepo Turborepo. Funkcje: uwierzytelnianie, płatności, panel administracyjny, i18n, CMS oparty na Git.

Szczegóły:
- Node.js >= 20.19.0
- pnpm z Turborepo
- Next.js 16 z React 19
- Drizzle ORM z PostgreSQL

## Plan sekcji

1. Instalacja - Skonfiguruj Node.js i sklonuj repozytorium
2. Konfiguracja środowiska - Skonfiguruj .env.local
3. Szybki start - Pierwszy serwer deweloperski
4. Pierwsze wdrożenie - Wdróż na Vercel lub Docker
5. Podręczny przewodnik - Ściągawka z poleceniami
