# Szybki start

## Krok 1: Konfiguracja
Edytuj apps/web/.content/config.yml po pierwszej synchronizacji.

## Krok 2: Skonfiguruj repozytorium danych
1. Utwórz fork https://github.com/ever-works/awesome-data
2. Ustaw DATA_REPOSITORY i GH_TOKEN w .env.local

## Krok 3: Uruchomienie
```bash
pnpm run dev
```

## Krok 4: Dodaj elementy
Utwórz pliki YAML w folderze items/ swojego repozytorium danych.
