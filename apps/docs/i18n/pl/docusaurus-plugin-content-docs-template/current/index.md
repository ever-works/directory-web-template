# Directory Web Template

Directory Web Template to nowoczesne, pełnostoikowe rozwiązanie do tworzenia stron katalogów, zbudowane na Next.js 16 i zorganizowane jako monorepo Turborepo. Zostało zaprojektowane, aby pomóc Ci tworzyć profesjonalne strony katalogów dla narzędzi, usług, produktów lub dowolnego innego rodzaju platformy z listingami.

## Główne funkcje

- **Nowoczesny stack technologiczny**: Next.js 16, React 19, TypeScript, Tailwind CSS, HeroUI React
- **Monorepo Turborepo**: przestrzenie robocze pnpm ze współdzielonymi konfiguracjami, aplikacją webową, testami e2e i dokumentacją
- **Elastyczne uwierzytelnianie**: NextAuth.js v5, Supabase Auth, dostawcy OAuth (Google, GitHub, Facebook, Twitter, Microsoft)
- **Integracja płatności**: Stripe, LemonSqueezy, Polar, zarządzanie subskrypcjami
- **Internacjonalizacja**: Wiele języków z pełną obsługą RTL via next-intl
- **CMS oparty na Git**: Synchronizacja treści z repozytoriów Git ze strukturą opartą na YAML
- **System motywów**: Wbudowane motywy z dynamicznym generowaniem kolorów
- **Analytics i monitorowanie**: PostHog, Sentry, monitorowanie wydajności
- **Panel administracyjny**: Zarządzanie treścią, użytkownikami i analityką
- **Zoptymalizowany pod SEO**: Generowanie mapy witryny, dane strukturalne (JSON-LD), metatagi

## Szybki start

```bash
git clone https://github.com/ever-works/directory-web-template.git
cd directory-web-template
pnpm install
pnpm run dev
```
