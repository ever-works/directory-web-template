# Przewodnik Szkolenia Zespołu

Kompleksowy program szkoleniowy do wdrożenia i doskonalenia zespołów deweloperskich na platformie Ever Works.

## 🎯 Cele Szkolenia

Po ukończeniu tego programu członkowie zespołu będą w stanie:

- ✅ Skonfigurować swoje środowisko deweloperskie
- ✅ Rozumieć architekturę Ever Works i stos technologiczny
- ✅ Przestrzegać ustalonych standardów i konwencji kodowania
- ✅ Dokumentować API za pomocą zautomatyzowanego systemu dokumentacji
- ✅ Implementować funkcje zgodnie z najlepszymi praktykami
- ✅ Debugować i rozwiązywać typowe problemy
- ✅ Pewnie wnosić wkład do kodu

## 📚 Moduły Szkoleniowe

### 1. [Wdrożenie](/team-training/onboarding)

**Czas trwania**: 1–2 dni

### 2. [Dokumentacja API](/team-training/api-documentation)

**Czas trwania**: 2–3 dni  
**Docelowi**: Wszyscy deweloperzy pracujący na endpointach API

### 3. [Najlepsze Praktyki](/team-training/best-practices)

**Czas trwania**: 1–2 dni  
**Docelowi**: Wszyscy członkowie zespołu

### 4. [Ćwiczenia Praktyczne](/team-training/exercises)

**Czas trwania**: 3–5 dni  
**Docelowi**: Wszyscy członkowie zespołu

## 📋 Wymagania Wstępne

### Wymagana Wiedza

- **JavaScript/TypeScript**: Średniozaawansowany do zaawansowanego
- **React**: Solidne zrozumienie hooków, komponentów, zarządzania stanem
- **Next.js**: Podstawowa znajomość (preferowany App Router)
- **Git**: Podstawy kontroli wersji

### Narzędzia do Zainstalowania

- Node.js 20.19.0+ (użyj nvm)
- pnpm ([instalacja](https://pnpm.io/installation))
- PostgreSQL 14+
- Git
- VS Code (lub preferowane IDE)

## 🆘 Uzyskiwanie Pomocy

**Podczas Szkolenia**:
- Zapytaj mentora lub lidera zespołu
- Skonsultuj [Przewodnik Szybkiej Pomocy](/getting-started/quick-reference)

**Po Szkoleniu**:
- Kanały Slack zespołu
- Opinie z code review
- Cotygodniowe spotkania zespołu
