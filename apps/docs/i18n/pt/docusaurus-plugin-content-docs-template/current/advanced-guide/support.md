## 🔧 Solução de problemas

### Problemas comuns

#### Problemas de instalação

- **Versão do Node.js**: verifique se você está usando o Node.js 18+
- **Dependências**: Execute `npm install` ou `yarn install` no diretório do site
- **Conflitos de porta**: Use o sinalizador `--port` para especificar uma porta diferente

#### Problemas de compilação

- **Erros TypeScript**: verifique sua configuração TypeScript
- **Dependências ausentes**: certifique-se de que todos os pacotes estejam instalados corretamente
- **Variáveis de ambiente**: Verifique a configuração do arquivo `.env` #### Problemas de tempo de execução

- **Conexão API**: Verifique sua variável de ambiente `NEXT_PUBLIC_API_BASE_URL` - **Autenticação**: verifique as credenciais do seu provedor OAuth
- **Banco de dados**: certifique-se de que sua conexão com o banco de dados esteja funcionando

### Modo de depuração

Habilite o registro de depuração definindo variáveis de ambiente:

```bash
DEBUG=directory-web-template:*
NODE_ENV=development
```
