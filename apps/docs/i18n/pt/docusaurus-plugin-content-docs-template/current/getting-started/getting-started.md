# Primeiros Passos com o Template

Bem-vindo ao Ever Works Directory Web Template. Do clone à aplicação em execução.

## O que é o Ever Works Template?

Site de diretório Next.js completo em um monorepo Turborepo. Funcionalidades: autenticação, pagamentos, painel administrativo, i18n, CMS baseado em Git.

Detalhes:
- Node.js >= 20.19.0
- pnpm com Turborepo
- Next.js 16 com React 19
- Drizzle ORM com PostgreSQL

## Roteiro da Seção

1. Instalação - Configurar Node.js e clonar o repositório
2. Configuração do Ambiente - Configurar .env.local
3. Início Rápido - Primeiro servidor de desenvolvimento
4. Primeiro Deploy - Deploy no Vercel ou Docker
