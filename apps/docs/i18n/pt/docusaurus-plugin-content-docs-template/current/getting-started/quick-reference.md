# Referência Rápida

## Comandos

```bash
pnpm run dev       # Iniciar servidores de desenvolvimento
pnpm run build     # Compilar tudo
pnpm run lint      # Lint em tudo
pnpm db:generate   # Gerar migrações
pnpm db:migrate    # Aplicar migrações
pnpm db:seed       # Popular dados
```

## Arquivos Principais

| Arquivo | Finalidade |
|---------|------------|
| apps/web/.env.local | Variáveis de ambiente |
| apps/web/.content/ | Conteúdo do CMS |
| apps/web/app/ | Rotas Next.js |
| apps/web/lib/ | Lógica de negócio |
