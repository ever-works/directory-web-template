# Início Rápido

## Passo 1: Configurar
Edite apps/web/.content/config.yml após o primeiro sync.

## Passo 2: Configurar o Repositório de Dados
1. Faça um fork de https://github.com/ever-works/awesome-data
2. Defina DATA_REPOSITORY e GH_TOKEN no .env.local

## Passo 3: Iniciar
```bash
pnpm run dev
```

## Passo 4: Adicionar Itens
Crie arquivos YAML na pasta items/ do seu repositório de dados.
