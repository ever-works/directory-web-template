# Directory Web Template

O Directory Web Template é uma solução moderna e full-stack para criação de sites de diretório, construída com Next.js 16 e organizada como um monorepo Turborepo. Foi projetada para ajudá-lo a criar sites de diretório profissionais para ferramentas, serviços, produtos ou qualquer outro tipo de plataforma de listagens.

## Principais Recursos

- **Stack Tecnológico Moderno**: Next.js 16, React 19, TypeScript, Tailwind CSS, HeroUI React
- **Monorepo Turborepo**: workspaces pnpm com configurações compartilhadas, aplicação web, testes e2e e documentação
- **Autenticação Flexível**: NextAuth.js v5, Supabase Auth, provedores OAuth (Google, GitHub, Facebook, Twitter, Microsoft)
- **Integração de Pagamentos**: Stripe, LemonSqueezy, Polar, gerenciamento de assinaturas
- **Internacionalização**: Múltiplos idiomas com suporte completo a RTL via next-intl
- **CMS Baseado em Git**: Sincronização de conteúdo de repositórios Git com estrutura baseada em YAML
- **Sistema de Temas**: Temas integrados com geração dinâmica de cores
- **Analytics & Monitoramento**: PostHog, Sentry, monitoramento de performance
- **Painel Administrativo**: Gerenciamento de conteúdo, usuários e analytics
- **SEO Otimizado**: Geração de sitemap, dados estruturados (JSON-LD), meta tags

## Início Rápido

```bash
git clone https://github.com/ever-works/directory-web-template.git
cd directory-web-template
pnpm install
pnpm run dev
```
