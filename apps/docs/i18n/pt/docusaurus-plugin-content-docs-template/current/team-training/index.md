# Guia de Treinamento da Equipe

Um programa de treinamento abrangente para integrar e capacitar equipes de desenvolvimento na plataforma Ever Works.

## 🎯 Objetivos do Treinamento

Ao final deste programa, os membros da equipe serão capazes de:

- ✅ Configurar seu ambiente de desenvolvimento
- ✅ Compreender a arquitetura e stack tecnológico do Ever Works
- ✅ Seguir padrões e convenções de codificação estabelecidos
- ✅ Documentar APIs com o sistema de documentação automatizado
- ✅ Implementar funcionalidades seguindo melhores práticas
- ✅ Depurar e resolver problemas comuns
- ✅ Contribuir para o codebase com confiança

## 📚 Módulos de Treinamento

### 1. [Integração](/team-training/onboarding)

**Duração**: 1–2 dias  
**Público**: Novos membros da equipe

### 2. [Documentação API](/team-training/api-documentation)

**Duração**: 2–3 dias  
**Público**: Todos os desenvolvedores que trabalham em endpoints de API

### 3. [Melhores Práticas](/team-training/best-practices)

**Duração**: 1–2 dias  
**Público**: Todos os membros da equipe

### 4. [Exercícios Práticos](/team-training/exercises)

**Duração**: 3–5 dias  
**Público**: Todos os membros da equipe

## 📅 Cronograma de Treinamento

### Novos Membros da Equipe (2 semanas)

**Semana 1: Fundamentos**
- Dias 1–2: [Integração](/team-training/onboarding)
- Dias 3–4: [Documentação API](/team-training/api-documentation)
- Dia 5: [Melhores Práticas](/team-training/best-practices)

**Semana 2: Prática**
- Dias 1–3: [Exercícios Práticos](/team-training/exercises)
- Dias 4–5: Primeira tarefa real com mentoria

## 📋 Pré-requisitos

### Conhecimento Necessário

- **JavaScript/TypeScript**: Intermediário a avançado
- **React**: Boa compreensão de hooks, componentes, gerenciamento de estado
- **Next.js**: Familiaridade básica (App Router preferido)
- **Git**: Noções básicas de controle de versão

### Ferramentas para Instalar

- Node.js 20.19.0+ (use nvm)
- pnpm ([instalação](https://pnpm.io/installation))
- PostgreSQL 14+
- Git
- VS Code (ou IDE preferido)

## 🆘 Obtendo Ajuda

**Durante o Treinamento**:
- Pergunte ao mentor ou líder técnico
- Consulte o [Guia de Referência Rápida](/getting-started/quick-reference)

**Após o Treinamento**:
- Canais do Slack da equipe
- Feedback de revisão de código
- Reuniões semanais da equipe
