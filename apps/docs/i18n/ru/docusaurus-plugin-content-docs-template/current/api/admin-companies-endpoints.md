# Конечные точки API компаний администратора

API компаний администратора предоставляет конечные точки для управления записями компаний. Поддерживает CRUD-операции с Zod-валидацией, проверкой уникальности домена/слага и необязательной синхронизацией с CRM.

## Сводка маршрутов

| Метод | Путь | Аутентификация | Описание |
|--------|------|------|-------------|
| `GET` | `/api/admin/companies` | Администратор | Список компаний (с пагинацией, поиском) |
| `POST` | `/api/admin/companies` | Администратор | Создать новую компанию |
| `GET` | `/api/admin/companies/{id}` | Администратор | Получить компанию по UUID |
| `PUT` | `/api/admin/companies/{id}` | Администратор | Обновить компанию |
| `DELETE` | `/api/admin/companies/{id}` | Администратор | Постоянно удалить компанию |

## Конечные точки

### GET `/api/admin/companies`

Список компаний с пагинацией, поиском и фильтрацией по статусу. Также возвращает глобальные счётчики активных/неактивных компаний.

**Параметры запроса:**

| Параметр | Тип | По умолчанию | Описание |
|-----------|------|---------|-------------|
| `page` | integer | `1` | Номер страницы |
| `limit` | integer | `10` | Элементов на странице (1–100) |
| `q` | string | — | Поиск по названию или домену |
| `status` | string | — | `"active"` или `"inactive"` |

### POST `/api/admin/companies`

Создаёт новую компанию. Данные валидируются через Zod-схему. Домен и слаг нормализуются до нижнего регистра. Проверяется уникальность домена и слага.

| Поле | Тип | Обязательный | Описание |
|-------|------|----------|-------------|
| `name` | string | Да | Название компании (1–255 симв.) |
| `website` | string | Нет | URL сайта |
| `domain` | string | Нет | Домен |
| `slug` | string | Нет | URL-слаг |
| `status` | string | Нет | `"active"` (default) / `"inactive"` |

### GET `/api/admin/companies/{id}`

Возвращает одну компанию по её UUID.

### PUT `/api/admin/companies/{id}`

Частичное обновление. После успешного обновления компания синхронизируется с CRM (неблокирующая операция).

### DELETE `/api/admin/companies/{id}`

Постоянно удаляет компанию. Действие необратимо. Связи элементов с компанией удаляются через CASCADE.

:::caution
Удаление компании необратимо. Все связи элементов будут удалены через CASCADE.
:::

## Коды ошибок

| Статус | Причина |
|--------|-------|
| `400` | Ошибка валидации |
| `401` | Отсутствует сессия администратора |
| `404` | Компания не найдена |
| `409` | Конфликт уникальности домена или слага |
| `500` | Внутренняя ошибка |
