# Конечные точки API элементов администратора

API элементов предоставляет конечные точки для управления записями каталога, видной очереди рецензий (одобрение/отклонение), историей аудита, групповыми операциями и статистикой. Статусы: `draft`, `pending`, `approved`, `rejected`.

## Базовый путь

```
/api/admin/items
```

## Сводка маршрутов

| Метод | Путь | Аутентификация | Описание |
|--------|-----|-------|-------------|
| `GET` | `/api/admin/items` | Администратор | Список элементов |
| `POST` | `/api/admin/items` | Администратор | Создать элемент |
| `GET` | `/api/admin/items/stats` | Администратор | Статистика |
| `POST` | `/api/admin/items/bulk` | Администратор | Групповые операции |
| `GET` | `/api/admin/items/{id}` | Администратор | Получить по ID |
| `PUT` | `/api/admin/items/{id}` | Администратор | Обновить |
| `DELETE` | `/api/admin/items/{id}` | Администратор | Удалить |
| `POST` | `/api/admin/items/{id}/review` | Администратор | Проверка |
| `GET` | `/api/admin/items/{id}/history` | Администратор | История аудита |

---

## Список элементов

```
GET /api/admin/items
```

**Параметры запроса:**

| Параметр | Тип | По умолчанию | Описание |
|-----------|------|---------|-------------|
| `page` | integer | `1` | Номер страницы |
| `limit` | integer | `10` | Элементов на странице (1–100) |
| `search` | string | — | Поиск по названию/описанию |
| `status` | string | — | `draft`, `pending`, `approved`, `rejected` |
| `categories` | string | — | Слаги через запятую |
| `tags` | string | — | Теги через запятую |
| `sortBy` | string | `updated_at` | `name`, `updated_at`, `status`, `submitted_at` |
| `sortOrder` | string | `desc` | `asc` или `desc` |

---

## Создать элемент

```
POST /api/admin/items
```

| Поле | Тип | Обязательный | Описание |
|-------|------|----------|-------------|
| `id` | string | Да | Уникальный идентификатор |
| `name` | string | Да | Имя элемента |
| `slug` | string | Да | URL-слаг (должен быть уникальным) |
| `description` | string | Да | Описание |
| `source_url` | string | Да | URL источника |
| `category` | string[] | Нет | Массив слагов категорий |
| `tags` | string[] | Нет | Массив слагов тегов |
| `status` | string | Нет | Начальный статус (default: `draft`) |

---

## Групповые операции

```
POST /api/admin/items/bulk
```

| Поле | Тип | Обязательный | Описание |
|-------|------|----------|-------------|
| `action` | string | Да | `approve`, `reject` или `delete` |
| `ids` | string[] | Да | ID элементов (1–100) |
| `reason` | string | Да (для reject) | Причина отклонения (мин. 10 симв.) |

---

## Проверка элемента

```
POST /api/admin/items/{id}/review
```

| Поле | Тип | Обязательный | Описание |
|-------|------|----------|-------------|
| `status` | string | Да | `approved` или `rejected` |
| `review_notes` | string | Нет | Пояснение решения |

---

## История аудита

```
GET /api/admin/items/{id}/history
```

Полный аудит элемента: создание, обновления, смена статуса, проверки, удаления, восстановления.

**Параметры запроса:**

| Параметр | Тип | Описание |
|-----------|------|-------------|
| `page` | integer | Номер страницы |
| `limit` | integer | Элементов на странице (max 100) |
| `action` | string | Фильтр по типу действия |

## Коды ошибок

| Статус | Значение |
|--------|----------|
| `400` | Ошибка валидации |
| `401` | Требуется аутентификация |
| `403` | Требуются права администратора |
| `404` | Элемент не найден |
| `409` | Дублирующийся ID или слаг |
| `500` | Внутренняя ошибка |
