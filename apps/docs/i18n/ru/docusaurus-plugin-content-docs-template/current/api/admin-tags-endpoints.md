# Конечные точки API тегов администратора

API тегов администратора предоставляет полные CRUD-операции для управления тегами контента. Теги используются для классификации и фильтрации элементов. Все операции записи сбрасывают кэш.

## Сводка маршрутов

| Метод | Путь | Аутентификация | Описание |
|--------|------|------|-------------|
| `GET` | `/api/admin/tags` | Администратор | Список (с пагинацией) |
| `POST` | `/api/admin/tags` | Администратор | Создать тег |
| `GET` | `/api/admin/tags/all` | Администратор | Все теги (из кэша) |
| `GET` | `/api/admin/tags/{id}` | Администратор | Получить по ID |
| `PUT` | `/api/admin/tags/{id}` | Администратор | Обновить |
| `DELETE` | `/api/admin/tags/{id}` | Администратор | Удалить |

## Конечные точки

### GET `/api/admin/tags`

**Параметры запроса:**

| Параметр | Тип | По умолчанию | Описание |
|-----------|------|---------|-------------|
| `page` | integer | `1` | Номер страницы |
| `limit` | integer | `10` | Элементов на странице (1–100) |

### POST `/api/admin/tags`

| Поле | Тип | Обязательный | Описание |
|-------|------|----------|-------------|
| `id` | string | Да | URL-слаг |
| `name` | string | Да | Отображаемое имя (2–50 симв.) |
| `isActive` | boolean | Нет | Активен ли (default: `true`) |

### GET `/api/admin/tags/all`

parameter `locale` (default: `"en"`) — код локали. Читает из кэша контента, не из базы данных.

### PUT `/api/admin/tags/{id}`

Обновляет имя и/или активность тега. ID тега нельзя изменить.

### DELETE `/api/admin/tags/{id}`

Постоянно удаляет тег. Также удаляются все связи тега с элементами.

:::caution
Удаление тега необратимо. Рассмотрите деактивацию (установка `isActive: false`), если хотите сохранить целостность данных.
:::

## Модель данных тега

| Поле | Тип | Описание |
|-------|------|-------------|
| `id` | string | URL-слаг |
| `name` | string | Отображаемое имя |
| `isActive` | boolean | Активен ли |
| `itemCount` | integer | Количество элементов |

## Коды ошибок

| Статус | Причина |
|--------|-------|
| `400` | Отсутствуют ID/имя; длина имени; неверная пагинация |
| `401` | Нет сессии |
| `404` | Тег не найден |
| `409` | Дублирующийся ID или имя |
| `500` | Внутренняя ошибка |
