# Конечные точки клиентского API

Конечные точки API, ориентированные на клиента, обслуживают прошедших проверку подлинности конечных пользователей (не администраторов). Эти маршруты обрабатывают панель управления клиента, отправку элементов, управление избранным и взаимодействие с общедоступными элементами, например комментарии, голоса и просмотры.

## Панель управления клиента и элементы (`/api/client`)

Для всех маршрутов `/api/client/*` требуется аутентифицированный сеанс с действительным `clientProfileId`.

### Панель управления

|Метод|Путь|Описание|
|--------|------|-------------|
|`GET`|`/api/client/dashboard/stats`|Статистика клиентской панели (количество товаров, просмотры, вовлеченность)|

### Клиентские элементы

|Метод|Путь|Описание|
|--------|------|-------------|
|`GET`|`/api/client/items`|Список элементов, отправленных текущим клиентом|
|`POST`|`/api/client/items`|Отправьте новый товар на рассмотрение|
|`GET`|`/api/client/items/stats`|Статистика по клиентским позициям (опубликовано, ожидается, отклонено)|
|`GET`|`/api/client/items/coordinates`|Получить координаты товаров клиента|
|`GET`|`/api/client/items/[id]`|Получить подробную информацию о товаре|
|`PUT`|`/api/client/items/[id]`|Обновить собственный элемент|
|`DELETE`|`/api/client/items/[id]`|Удалить собственный элемент (обратимое удаление)|
|`POST`|`/api/client/items/[id]/restore`|Восстановление обратимо удаленного элемента|

### Географическая статистика

|Метод|Путь|Описание|
|--------|------|-------------|
|`GET`|`/api/client/geo-stats`|Географическая статистика по клиентским товарам|

## Взаимодействие с общедоступными объектами (`/api/items`)

Эти конечные точки обрабатывают функции общедоступных элементов. Некоторые требуют аутентификации (например, голосование), тогда как другие полностью общедоступны (например, просмотр).

### Комментарии

|Метод|Путь|Описание|Авторизация|
|--------|------|-------------|------|
|`GET`|`/api/items/[slug]/comments`|Список комментариев к элементу|Общественный|
|`POST`|`/api/items/[slug]/comments`|Добавить комментарий|Требуется|
|`GET`|`/api/items/[slug]/comments/[commentId]`|Получить подробную информацию о комментариях|Общественный|
|`PUT`|`/api/items/[slug]/comments/[commentId]`|Обновить собственный комментарий|Требуется|
|`DELETE`|`/api/items/[slug]/comments/[commentId]`|Удалить собственный комментарий|Требуется|

### Рейтинг комментариев

|Метод|Путь|Описание|Авторизация|
|--------|------|-------------|------|
|`GET`|`/api/items/[slug]/comments/rating`|Получить сводку рейтинга|Общественный|
|`POST`|`/api/items/[slug]/comments/rating`|Отправить оценку|Требуется|
|`GET`|`/api/items/[slug]/comments/rating/[commentId]`|Получить оценку за комментарий|Общественный|

### Голоса

|Метод|Путь|Описание|Авторизация|
|--------|------|-------------|------|
|`GET`|`/api/items/[slug]/votes/count`|Получить подсчет голосов|Общественный|
|`GET`|`/api/items/[slug]/votes/status`|Получить статус голосования текущего пользователя|Требуется|
|`POST`|`/api/items/[slug]/votes`|Проголосовать за предмет (за/против)|Требуется|

### Просмотры

|Метод|Путь|Описание|Авторизация|
|--------|------|-------------|------|
|`POST`|`/api/items/[slug]/views`|Запись просмотра страницы|Общественный|

### Вовлеченность и популярность

|Метод|Путь|Описание|Авторизация|
|--------|------|-------------|------|
|`GET`|`/api/items/engagement`|Получите показатели вовлеченности для элементов|Общественный|
|`GET`|`/api/items/popularity-scores`|Получите рассчитанные оценки популярности|Общественный|

### Компания

|Метод|Путь|Описание|Авторизация|
|--------|------|-------------|------|
|`GET`|`/api/items/[slug]/company`|Получить информацию о компании для товара|Общественный|

## Избранное (`/api/favorites`)

Управляйте любимыми предметами пользователя. Все конечные точки избранного требуют аутентификации.

|Метод|Путь|Описание|
|--------|------|-------------|
|`GET`|`/api/favorites`|Список любимых предметов текущего пользователя|
|`POST`|`/api/favorites/[itemSlug]`|Переключить статус избранного для элемента|
|`DELETE`|`/api/favorites/[itemSlug]`|Удалить товар из избранного|

## Текущий пользователь (`/api/current-user`)

## Спонсорские объявления – пользователь (`/api/sponsor-ads/user`)

|Метод|Путь|Описание|
|--------|------|-------------|
|`GET`|`/api/sponsor-ads/user`|Получение списка спонсорских объявлений пользователя|
|`GET`|`/api/sponsor-ads/user/stats`|Статистика эффективности рекламы пользователей|
|`GET`|`/api/sponsor-ads/user/[id]`|Получить подробную информацию об объявлении|
|`PUT`|`/api/sponsor-ads/user/[id]`|Обновить собственное объявление|
|`POST`|`/api/sponsor-ads/user/[id]/cancel`|Отменить собственное объявление|
|`POST`|`/api/sponsor-ads/user/[id]/renew`|Продлить объявление с истекшим сроком действия|

|Метод|Путь|Описание|Авторизация|

## Конечные точки общедоступных данных

Эти конечные точки не требуют аутентификации:

|Метод|Путь|Описание|
|--------|------|-------------|
|`GET`|`/api/categories/exists`|Проверьте, существует ли ярлык категории|
|`GET`|`/api/collections/exists`|Проверьте, существует ли пул сбора|
|`GET`|`/api/featured-items`|Список избранных товаров|
|`GET`|`/api/sponsor-ads`|Получите активную спонсорскую рекламу для показа|
|`POST`|`/api/sponsor-ads/checkout`|Начать оформление заказа спонсорской рекламы|

## Шаблоны пагинации

Конечные точки клиентского списка поддерживают стандартные параметры нумерации страниц:

```
GET /api/client/items?page=1&limit=10&sort=createdAt&order=desc
GET /api/items/[slug]/comments?page=1&limit=20
GET /api/favorites?page=1&limit=50
```

Ответы включают метаданные нумерации страниц:

```json
{
  "success": true,
  "data": [...],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 45,
    "totalPages": 5
  }
}
```
