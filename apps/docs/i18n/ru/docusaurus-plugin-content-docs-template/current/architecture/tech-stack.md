# Технологический стек

В этом документе представлен всеобъемлющий обзор всех технологий, используемых в Ever Works.

## Системные требования

- **Node.js**: 20.19.0 или выше
- **PostgreSQL**: 14.0 или выше
- **Менеджер пакетов**: npm, pnpm, пряжа или булочка.

## Фронтенд-технологии {#frontend}

### Основная структура

- **[Next.js 15.4.7](https://nextjs.org/)** — платформа React с App Router
  - Серверный рендеринг (SSR)
  - Генерация статического сайта (SSG)
  - Инкрементная статическая регенерация (ISR)
  - Действия сервера для мутаций
  - Встроенная оптимизация
  - Маршрутизация на основе файлов с динамическими сегментами `[locale]`

- **[React 19.1.0](https://react.dev/)** - библиотека пользовательского интерфейса
  - Новейшие функции и улучшения
  - Параллельный рендеринг
  - Автоматическое дозирование
  - Приостановка получения данных
  - Серверные компоненты по умолчанию

### Язык и безопасность типов

- **[TypeScript 5.x](https://www.typescriptlang.org/)** - Статическая проверка типов
  - Строгий режим включен
  - Сопоставление путей настроено (`@/` псевдоним)
  - Определения пользовательских типов
  - Полный вывод типа

### Стиль и пользовательский интерфейс

- **[Tailwind CSS 3.4](https://tailwindcss.com/)** — CSS-фреймворк, ориентированный на утилиты
  - Система индивидуального дизайна
  - Поддержка темного режима
  - Утилиты адаптивного дизайна
  - JIT-компиляция
  - Динамическая система цвета (50-950 оттенков)

- **[HeroUI React 2.6](https://www.heroui.com/)** — Современные компоненты React
  - Доступные компоненты
  - Настраиваемые темы
  - Поддержка TypeScript
  - Дерево-трясущееся

- **[Radix UI](https://www.radix-ui.com/)** — доступные компоненты без стилей
  - Безголовые примитивы пользовательского интерфейса
  - Полная навигация с помощью клавиатуры
  - Соответствие ARIA
  - Сборный

- **[Framer Motion 12.x](https://www.framer.com/motion/)** - Библиотека анимации
  - Декларативная анимация
  - Поддержка жестов
  - Анимация макета
  - SVG-анимация

### Редактирование расширенного текста

- **[TipTap](https://tiptap.dev/)** - Безголовый редактор форматированного текста
  - Расширяемая архитектура

## Бэкэнд-технологии

### База данных и ORM

- **[PostgreSQL 14+](https://www.postgresql.org/)** - Реляционная база данных
  - соответствие требованиям ACID
  - Расширенные функции (JSONB, полнотекстовый поиск)
  - Отличная производительность
  - Поддержка JSON
  - Триггеры и хранимые процедуры

- **[Drizzle ORM 0.40.0](https://orm.drizzle.team/)** - TypeScript ORM
  - Типобезопасные запросы
  - Минимальные накладные расходы
  - SQL-подобный синтаксис
  - Миграционная система
  - Запросы отношений
  - Подготовленные заявления

- **[Supabase](https://supabase.com/)** — серверная часть как услуга (необязательно)
  - Хостинг PostgreSQL
  - Подписки в реальном времени
  - Безопасность на уровне строк
  - Встроенная авторизация
  - Ведра для хранения
  - Краевые функции

### Аутентификация

- **[NextAuth.js 5.0 (бета)](https://authjs.dev/)** — библиотека аутентификации
  - Несколько поставщиков OAuth (Google, GitHub, Facebook, Twitter)
  - JWT и сеансы базы данных
  - Поддержка TypeScript
  - Лучшие практики безопасности
  - Авторизация на основе учетных данных
  - Управление сеансами

- **[Supabase Auth](https://supabase.com/auth)** – альтернативное решение для аутентификации.
  - Встроенное управление пользователями
  - Социальные провайдеры
  - Проверка электронной почты
  - Сброс пароля
  - Волшебные ссылки
  - Авторизация по телефону

### Архитектура двойной аутентификации

Ever Works поддерживает **как NextAuth.js, так и Supabase Auth** одновременно:

- NextAuth для традиционных потоков OAuth
- Supabase Auth для функций реального времени
- Единое управление сеансами
- Бесшовное переключение провайдера

## Требования к версии

### Node.js

- **Минимум**: Node.js 20.19.0.
- **Рекомендуется**: последняя версия LTS.
- **Менеджер пакетов**: npm 10+, Yarn 1.13+ или pnpm 8+.

### Поддержка браузера

- **Современные браузеры**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+.
- **Мобильные устройства**: iOS Safari 14+, Chrome Mobile 90+.
- **Нет поддержки IE**: только современные функции.

- **Встряхивание дерева**: удаление неиспользуемого кода.
- **Динамический импорт**: отложенная загрузка некритических компонентов.

- **Оптимизация изображений**: поддержка WebP/AVIF с отложенной загрузкой.
- **Оптимизация шрифтов**: автономные шрифты с предварительной загрузкой.

### Производительность базы данных

- **Пул соединений**: эффективные подключения к базе данных
- **Оптимизация запросов**: индексированные запросы и эффективные соединения.
- **Кэширование**: Кэширование на уровне приложения и на уровне базы данных.

|Требование|Выбор технологии|Обоснование|
