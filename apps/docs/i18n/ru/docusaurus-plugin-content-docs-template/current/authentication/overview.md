# Обзор Аутентификации

Ever Works предоставляет гибкую, безопасную систему аутентификации, поддерживающую множество провайдеров и методов аутентификации.

## Архитектура Аутентификации

Шаблон использует гибридный подход к аутентификации, поддерживая одновременно NextAuth.js и Supabase Auth, что позволяет выбрать лучшее решение для ваших нужд.

## Поддерживаемые Методы Аутентификации

### 1. OAuth-провайдеры

NextAuth.js OAuth поддерживает: Google, GitHub, Facebook, Twitter/X, Microsoft
Supabase OAuth поддерживает: Google, GitHub, Facebook, Twitter/X, Discord, Apple

### 2. Аутентификация по Email/Паролю

NextAuth.js Credentials: Пользовательский email/пароль, хэширование bcrypt, хранение сессий в базе данных
Supabase Auth: Встроенный email/пароль, верификация email, сброс пароля

### 3. Magic Links
Supabase Auth: Беспарольная аутентификация через magic-ссылки по email

### 4. WebAuthn / Passkeys
NextAuth.js: Биометрическая аутентификация, аппаратные ключи безопасности, FIDO2

## Управление Сессиями
JWT-токены для аутентификации без сохранения состояния, сессии базы данных для постоянного состояния, безопасная обработка cookie, автоматическое обновление токенов

## Функции Безопасности
Защита от CSRF, ограничение частоты запросов, защита от перебора, безопасное хэширование паролей с bcrypt
