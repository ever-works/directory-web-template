# Управление Сессиями

## Стратегия Сессий

Шаблон поддерживает две стратегии сессий:
1. JWT (по умолчанию) — Без сохранения состояния, хранится в cookie
2. База данных — Хранится в базе данных, поддерживает отзыв

## Конфигурация Сессии

```typescript
// auth.config.ts
export const authConfig = {
  session: {
    strategy: 'jwt',
    maxAge: 30 * 24 * 60 * 60, // 30 дней
  }
}
```

## Безопасность

- HttpOnly cookie предотвращают XSS
- SameSite=Lax предотвращает CSRF
- Автоматическое обновление сессии
- Флаг Secure в production

## Выход из системы

Сессия очищается при выходе. Все активные сессии можно аннулировать, изменив AUTH_SECRET.
