# Часто задаваемые вопросы

## Общее

### Что такое Directory Web Template?
Готовое к производству полнофункциональное решение для сайтов-каталогов на Next.js. Клонируйте, настройте и разверните для создания профессиональных сайтов-каталогов.

### Можно ли использовать Template без платформы Ever Works?
Да. Template работает независимо. Платформа — это отдельный опциональный продукт.

## Технологический стек

### Какие технологии использует Template?
- Фреймворк: Next.js 15, React 19
- Язык: TypeScript 5
- Стилизация: Tailwind CSS 4, HeroUI React
- ORM: Drizzle ORM с PostgreSQL
- Auth: NextAuth.js v5
- Платежи: Stripe, LemonSqueezy, Polar

### Какие провайдеры аутентификации поддерживаются?
Google, GitHub, Facebook, Twitter и Microsoft через NextAuth.js v5.

### Какие платёжные провайдеры поддерживаются?
Stripe, LemonSqueezy и Polar.

## Развёртывание

### Как развернуть Template?
Рекомендуется: Vercel. Docker также поддерживается.

### Какую базу данных использовать?
PostgreSQL, как правило через Supabase или прямое подключение.

## Контент

### Как работает CMS на основе Git?
Контент хранится в файлах YAML/Markdown в Git-репозитории, который клонируется во время сборки в .content/.

### Можно ли добавлять элементы вручную?
Да, редактируйте файлы YAML/Markdown в репозитории данных CMS.
