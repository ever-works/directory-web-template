# Начало работы с Template

Добро пожаловать в Ever Works Directory Web Template. От клонирования до запущенного приложения.

## Что такое Ever Works Template?

Полнофункциональный сайт-каталог Next.js в монорепозитории Turborepo. Возможности: аутентификация, платежи, панель администратора, i18n, CMS на основе Git.

Детали:
- Node.js >= 20.19.0
- pnpm с Turborepo
- Next.js 16 с React 19
- Drizzle ORM с PostgreSQL

## Дорожная карта раздела

1. Установка — настройка Node.js и клонирование репозитория
2. Настройка среды — конфигурация .env.local
3. Быстрый старт — первый сервер разработки
4. Первое развёртывание — deploy на Vercel или Docker
5. Краткий справочник — шпаргалка по командам
