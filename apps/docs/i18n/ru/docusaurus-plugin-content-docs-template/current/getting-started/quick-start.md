# Быстрый старт

## Шаг 1: Настройка
Отредактируйте apps/web/.content/config.yml после первой синхронизации.

## Шаг 2: Настройте репозиторий данных
1. Сделайте форк https://github.com/ever-works/awesome-data
2. Укажите DATA_REPOSITORY и GH_TOKEN в .env.local

## Шаг 3: Запуск
```bash
pnpm run dev
```

## Шаг 4: Добавление элементов
Создайте файлы YAML в папке items/ вашего репозитория данных.
