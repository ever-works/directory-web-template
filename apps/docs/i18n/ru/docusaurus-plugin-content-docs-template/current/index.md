# Directory Web Template

Directory Web Template — это современное полнофункциональное решение для создания сайтов-каталогов, построенное на Next.js 16 и организованное в виде монорепозитория Turborepo. Оно разработано для создания профессиональных сайтов-каталогов инструментов, сервисов, продуктов или любых других платформ с листингами.

## Основные функции

- **Современный технологический стек**: Next.js 16, React 19, TypeScript, Tailwind CSS, HeroUI React
- **Монорепозиторий Turborepo**: рабочие пространства pnpm с общими конфигурациями, веб-приложением, e2e-тестами и документацией
- **Гибкая аутентификация**: NextAuth.js v5, Supabase Auth, провайдеры OAuth (Google, GitHub, Facebook, Twitter, Microsoft)
- **Интеграция платежей**: Stripe, LemonSqueezy, Polar, управление подписками
- **Интернационализация**: Несколько языков с полной поддержкой RTL через next-intl
- **CMS на основе Git**: Синхронизация контента из Git-репозиториев со структурой на основе YAML
- **Система тем**: Встроенные темы с динамической генерацией цветов
- **Аналитика и мониторинг**: PostHog, Sentry, мониторинг производительности
- **Панель администратора**: Управление контентом, пользователями и аналитикой
- **SEO-оптимизация**: Генерация карты сайта, структурированные данные (JSON-LD), мета-теги

## Быстрый старт

```bash
git clone https://github.com/ever-works/directory-web-template.git
cd directory-web-template
pnpm install
pnpm run dev
```
