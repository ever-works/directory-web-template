# Руководство по Обучению Команды

Комплексная программа обучения для введения в работу и повышения квалификации команд разработчиков на платформе Ever Works.

## 🎯 Цели обучения

По завершении программы члены команды смогут:

- ✅ Настроить рабочую среду разработки
- ✅ Понимать архитектуру Ever Works и технологический стек
- ✅ Следовать установленным стандартам и соглашениям написания кода
- ✅ Документировать API с помощью автоматизированной системы документации
- ✅ Реализовывать функции согласно лучшим практикам
- ✅ Отлаживать и решать типичные проблемы
- ✅ Уверенно вносить вклад в кодовую базу

## 📚 Учебные модули

### 1. [Введение в работу](/team-training/onboarding)

**Продолжительность**: 1–2 дня  
**Аудитория**: Новые члены команды

### 2. [Документация API](/team-training/api-documentation)

**Продолжительность**: 2–3 дня  
**Аудитория**: Все разработчики, работающие с API-эндпоинтами

### 3. [Лучшие практики](/team-training/best-practices)

**Продолжительность**: 1–2 дня  
**Аудитория**: Все члены команды

### 4. [Практические упражнения](/team-training/exercises)

**Продолжительность**: 3–5 дней  
**Аудитория**: Все члены команды

## 📅 Расписание обучения

### Новые члены команды (2 недели)

**Неделя 1: Основы**
- Дни 1–2: [Введение в работу](/team-training/onboarding)
- Дни 3–4: [Документация API](/team-training/api-documentation)
- День 5: [Лучшие практики](/team-training/best-practices)

**Неделя 2: Практика**
- Дни 1–3: [Практические упражнения](/team-training/exercises)
- Дни 4–5: Первое реальное задание с наставником

## 📋 Предварительные требования

### Необходимые знания

- **JavaScript/TypeScript**: Средний и выше уровень
- **React**: Хорошее понимание хуков, компонентов, управления состоянием
- **Next.js**: Базовое знакомство (предпочтителен App Router)
- **Git**: Основы контроля версий

### Инструменты для установки

- Node.js 20.19.0+ (используйте nvm)
- pnpm ([установка](https://pnpm.io/installation))
- PostgreSQL 14+
- Git
- VS Code (или предпочитаемая IDE)

## 🆘 Получение помощи

**Во время обучения**:
- Спросите наставника или тимлида
- Обратитесь к [Руководству быстрого доступа](/getting-started/quick-reference)

**После обучения**:
- Каналы Slack команды
- Обратная связь на код-ревью
- Еженедельные встречи команды
