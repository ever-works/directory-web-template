## 🔧 故障排除

### 常见问题

#### 安装问题

- **Node.js 版本**：确保您使用的是 Node.js 18+
- **依赖关系**：在网站目录中运行1或2
- **端口冲突**：使用 3 标志指定不同的端口

#### 构建问题

- **TypeScript 错误**：检查您的 TypeScript 配置
- **缺少依赖项**：确保所有软件包均已正确安装
- **环境变量**：验证您的 `.env` 文件配置

#### 运行时问题

- **API 连接**：检查您的 `NEXT_PUBLIC_API_BASE_URL` 环境变量
- **身份验证**：验证您的 OAuth 提供商凭据
- **数据库**：确保您的数据库连接正常工作

### 调试模式

通过设置环境变量启用调试日志记录：

```bash
DEBUG=directory-web-template:*
NODE_ENV=development
```
