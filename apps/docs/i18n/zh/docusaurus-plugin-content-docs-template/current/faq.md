# 常见问题

## 什么是 Directory Web Template？

Directory Web Template 是一个基于 Next.js 构建的生产就绪目录网站解决方案，专为启动工具、服务或产品目录而设计。

## 我可以在没有 Ever Works Platform 的情况下使用 Template 吗？

可以，模板完全独立于 Ever Works 平台运行。

## 技术栈是什么？

- **Next.js 15** – React 框架
- **React 19** – UI 库
- **TypeScript 5** – 类型化编程
- **Tailwind CSS** – CSS 框架
- **Drizzle ORM** – 数据库管理
- **NextAuth.js** – 身份验证

## 支持哪些 OAuth 提供商？

- Google
- GitHub
- Facebook
- Twitter
- Microsoft

## 支持哪些支付提供商？

- Stripe
- LemonSqueezy
- Polar

## 推荐的部署平台是什么？

推荐使用 **Vercel** 进行最简单快速的部署。

## 推荐使用哪种数据库？

生产环境推荐通过 Supabase 使用 **PostgreSQL**。

## CMS 系统如何工作？

内容通过克隆到 `.content/` 目录中 Git 仓库中的 YAML/Markdown 文件进行管理。
