# 模板快速开始

欢迎使用 **Directory Web Template**！本指南将带您从克隆仓库到运行应用程序。

## 什么是 Ever Works Template？

Ever Works Template 是一个功能完整的 Next.js 目录，组织在 Turborepo 单仓库中。提供启动工具、服务或产品目录所需的一切。

## 系统要求

- **Node.js >= 20.19.0**
- **pnpm** – 包管理器
- **Next.js 16** – React 框架
- **Drizzle ORM** – 数据库管理
