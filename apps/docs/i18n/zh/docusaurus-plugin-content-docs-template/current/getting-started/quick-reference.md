# 快速参考

## 常用命令

| 命令 | 描述 |
|------|------|
| `pnpm run dev` | 启动开发服务器 |
| `pnpm run build` | 构建生产版本 |
| `pnpm run lint` | 检查代码错误 |
| `pnpm run db:generate` | 生成数据库迁移 |
| `pnpm run db:migrate` | 应用迁移 |
| `pnpm run db:seed` | 加载初始数据 |

## 关键文件和目录

| 文件/目录 | 描述 |
|----------|------|
| `.env.local` | 本地环境变量 |
| `.content/` | 从 Git CMS 克隆的内容 |
| `app/` | Next.js App Router 路由 |
| `lib/` | 业务逻辑和服务 |
| `components/` | React 组件 |
