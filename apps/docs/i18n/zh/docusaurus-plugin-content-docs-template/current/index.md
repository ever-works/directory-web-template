# Directory Web Template

**Directory Web Template** 是一个基于 Next.js 16 构建的现代化全栈目录网站解决方案，专为工具、服务、产品或列表平台的专业目录网站而设计。

## 主要功能

- **现代技术栈** – Next.js 16、React 19、TypeScript
- **Turborepo 单仓库** – 有组织的项目结构
- **灵活的身份验证** – NextAuth.js v5、OAuth 提供商
- **支付集成** – Stripe、LemonSqueezy、Polar
- **国际化** – 多语言支持
- **基于 Git 的 CMS** – 通过 Git 管理内容
- **主题系统** – 自定义设计
- **分析与监控** – 内置分析功能
- **管理仪表板** – 内容管理
- **SEO 优化** – 针对搜索引擎优化

## 快速开始

```bash
git clone https://github.com/ever-works/directory-web-template.git
cd directory-web-template
pnpm install
pnpm run dev
```
