# 团队培训指南

Ever Works 平台开发团队的全面入职和技能提升培训计划。

## 🎯 培训目标

完成培训计划后，团队成员将能够：

- ✅ 搭建开发工作环境
- ✅ 理解 Ever Works 的架构和技术栈
- ✅ 遵循既定的代码标准和规范
- ✅ 使用自动化文档系统记录 API
- ✅ 按照最佳实践实现功能
- ✅ 调试和解决常见问题
- ✅ 自信地为代码库做出贡献

## 📚 培训模块

### 1. [入职指南](/team-training/onboarding)

**时长**：1–2 天  
**受众**：新团队成员

### 2. [API 文档](/team-training/api-documentation)

**时长**：2–3 天  
**受众**：所有使用 API 端点的开发者

### 3. [最佳实践](/team-training/best-practices)

**时长**：1–2 天  
**受众**：所有团队成员

### 4. [实践练习](/team-training/exercises)

**时长**：3–5 天  
**受众**：所有团队成员

## 📅 培训时间表

### 新团队成员（2 周）

**第 1 周：基础知识**
- 第 1–2 天：[入职指南](/team-training/onboarding)
- 第 3–4 天：[API 文档](/team-training/api-documentation)
- 第 5 天：[最佳实践](/team-training/best-practices)

**第 2 周：实践**
- 第 1–3 天：[实践练习](/team-training/exercises)
- 第 4–5 天：在导师指导下完成第一个真实任务

## 📋 先决条件

### 所需知识

- **JavaScript/TypeScript**：中级及以上
- **React**：对 Hooks、组件、状态管理有良好理解
- **Next.js**：基本了解（推荐 App Router）
- **Git**：版本控制基础知识

### 需要安装的工具

- Node.js 20.19.0+（使用 nvm）
- pnpm（[安装](https://pnpm.io/installation)）
- PostgreSQL 14+
- Git
- VS Code（或首选 IDE）

## 🆘 获取帮助

**培训期间**：
- 询问导师或技术负责人
- 查阅[快速参考指南](/getting-started/quick-reference)

**培训后**：
- 团队 Slack 频道
- 代码审查反馈
- 每周团队会议
