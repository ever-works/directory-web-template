# Ever Works Documentation

Welcome to the documentation for **Ever Works** — a modern directory website ecosystem consisting of the **Ever Works Platform**, the **Ever Works Website Template**, and the **Pinler.com** SaaS service.

## What is Ever Works?

Ever Works is a complete solution for creating and managing professional directory websites. It comprises three main components:

- **[Ever Works Platform](/platform)** — The backend platform that powers directory websites, featuring a NestJS API, Next.js web dashboard, CLI tools, AI agents, and a plugin system.
- **[Ever Works Website Template](/template)** — A production-ready, full-stack directory website template built with Next.js 15, React 19, TypeScript, and Tailwind CSS.
- **[Pinler.com](https://pinler.com)** — A SaaS service built on top of the Ever Works Platform and Template.
