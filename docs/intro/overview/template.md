# Ever Works Website Template

The **Ever Works Website Template** is a production-ready, full-stack directory website solution built with modern web technologies. It provides everything you need to launch a professional directory website quickly.

## Key Features

- **Modern Tech Stack** — Next.js 15, React 19, TypeScript, Tailwind CSS, HeroUI React
- **Flexible Authentication** — NextAuth.js v5, Supabase Auth, OAuth providers
- **Payment Integration** — Stripe, LemonSqueezy, Polar, subscription management
- **Internationalization** — Multiple languages with RTL support
- **Git-based CMS** — Content synchronization from Git repositories
- **Theming System** — Built-in themes with dynamic color generation
- **Analytics & Monitoring** — PostHog, Sentry, performance monitoring

## Use Cases

The template is ideal for tool directories, service marketplaces, resource catalogs, professional directories, product showcases, and community platforms.
